## 记忆曲线学习法应用指南

> 核心原理：赫尔曼·艾宾浩斯发现，学习后20分钟记忆保持率约58%，1小时后降至44%，31天后仅剩19%。通过在遗忘临界点复习，可显著延缓遗忘。

### 复习节奏设计

| 阶段 | 时间点 | 方式 | 适用科目 |
|---|---|---|---|
| **第1次** | 学习后 **5分钟** | 快速回想核心要点 | 古文、RE词汇、NC3语法 |
| **第2次** | 学习后 **30分钟** | 利用课间休息或切换科目前的间隙 | 所有背诵类 |
| **第3次** | 当天 **睡前** | 闭眼回想或快速口述（约12h后） | 古文、英语发音 |
| **第4次** | **次日** | 学习新内容前花3分钟快速回顾 | 所有科目 |
| **第5次** | **2天后** | 快速检测——能回忆多少？ | 古文、RE词汇 |
| **第6次** | **4天后** | 结合复习日进行综合回顾 | 古文（每周复习日） |
| **第7次** | **7天后** | 每周复盘时整体检测 | 全部科目 |

### 各科目具体应用

**古文背诵（最适用）**
- 当天：背新篇 → 5分钟后回想 → 30分钟后续背 → 睡前默背
- 次日：背诵新篇前先花3分钟回顾前一天的篇目
- 复习日（每周第7天）：复习本周全部6篇，重点回顾4天前背的篇目
- 利用Chris记忆力的优势，配合这个节奏可以在不增加时间的情况下大幅提高保持率

**RE词汇联想**
- 当天：学完5个单词 → 5分钟后造句 → 30分钟后再造句 → 睡前再看一遍
- 次日：学新词前回顾前一天的单词
- 2天后、4天后：在RE学习前花2分钟快速过一遍之前学过的单词

**NC3语法**
- 当天：学完语法点 → 5分钟后自己举例 → 睡前回想语法规则
- 次日：复习前一课的语法点，再学新课

**英语发音**
- 发音是肌肉记忆，适合"少量多次"的复习节奏
- 每天练习2-3个音素，在刷牙、走路等碎片时间回想发音口型

### 给家长的实操建议

1. **不要额外安排"复习时间"** — 将复习嵌入现有流程中，比如在切换科目前的间隙花1-2分钟快速回顾
2. **利用Chris的记忆力优势** — Chris每天可背约300字，配合记忆曲线，这个数字可能翻倍
3. **用小测验代替复习** — "昨天背的那篇古文，你还记得第一句是什么吗？"比"来，我们再背一遍"更有效
4. **打卡表上的复习日已经嵌入了这个节奏** — 每7天一个复习日，正好对应7天周期
5. **不追求100%回忆** — 如果某次复习只能回忆60%，也是正常的，继续按节奏走即可

# 目标一：练字（暑假最大任务：聚焦正确性与习惯养成）

## 现状分析

Chris的写字问题主要表现为：字整体向左歪、字体大小不一、笔画轻。此前已多次纠正，握笔口诀已熟记，坐姿要点也清楚，但一旦开始专注写内容就容易回到旧习惯。当前基线：能写出工整字但需要高度集中注意力，放松时字就歪。暑假需要从"刻意工整"过渡到"自然工整"。

**本暑假练字核心要求**：
1.  **写工整**：不要求写得十分美观，但必须工整、大小一致、清晰有条理。
2.  **写快**：在保证工整的前提下，逐步提高书写速度，以适应未来作业量。

> **💡 练字指导思想：工整训练 vs 快速训练**
> 练字分为两个场景，各司其职，不可混淆：
> - **专项练字时间**（每日2次各10分钟）→ 专注练**工整**。用米字格、中线训练等方法，只练正确性，不追求速度。
> - **应用书写时间**（古文默写、数学写步骤、英语抄写）→ 自然练**快速**。在这些科目中要求字迹可辨认即可，重点是练习在有一定时间压力下写出清晰的字。
> 
> 不必要求在练字时间内同时追求"又快又好"——那是矛盾的目标。分开训练，各得其所。

## 字体推荐：衡水体（考试快速书写体）

**推荐理由**：衡水体源自河北衡水中学，是目前国内最适合考试快速书写的字体。和田英章楷书不同——田英章是艺术楷书，追求每个字的美观但书写速度慢；衡水体是**考试书写体**，追求"清楚、整齐、快速"——不是好看，而是**不扣分、写得快**。这是暑假练字的最终目标。

**核心哲学**：书法家练字是"写好一个字练100遍"，考试练字是"100个字都写得差不多就行"。衡水体不追求每个字的艺术性，追求的是**整篇看起来干净整齐**。

### 衡水体 vs 田英章楷书 对比

| 维度 | 田英章楷书（传统练字） | 衡水体（考试书写） |
|---|---|---|
| **目标** | 每个字写得像字帖一样美 | 整篇看起来整齐清楚，不扣分 |
| **速度** | 慢（追求笔画完美） | **快**（追求整体效果） |
| **连笔** | 禁止连笔 | **允许轻微连笔**（提高速度的关键） |
| **单字** | 每个字独立完美 | 单字70分即可，整篇90分 |
| **练习方法** | 一个字写100遍 | 100个字各写1遍 |
| **适用场景** | 书法比赛、硬笔考级 | **日常作业、考试答卷** |
| **见效时间** | 3-6个月 | **2-4周**可见明显改善 |

### 字体核心要点（衡水体）

```
① 横平竖直（与楷体相同，但要求降低）
   - 横画：大致水平即可，允许轻微上斜（5°以内）
   - 竖画：垂直即可，不要歪就行
   - 核心：不要求像印刷体一样直，只要不歪得太明显
   - 练习：在米字格中画"十"字，各5遍

② 大小一致（最重要的得分点）
   - 每个字占格子的70-80%，**这是衡水体的灵魂**
   - 写字大的扣分，写字小的扣分，忽大忽小的扣分——大小一致能拿大半的分
   - 练习方法：不用练单字之美，练"一行字上下对齐、左右均匀"
   - 最快的练习方式：在方格本中抄写任何文字，只关注"大小一致"

③ 重心稳定（不是居中，而是稳定）
   - 核心不是"每个字都在正中央"，而是"所有字的底部在一条线上"
   - 衡水体的核心秘密：**字可以微偏，但底部必须齐平**
   - 练习方法：抄写时只看字的底线是否对齐，不看上面

④ 笔画简化（提高速度的关键）
   - 允许轻微连笔：如"口"字可以一笔写成（不抬笔）
   - 允许简省：如"的"字的撇可以简化
   - 允许行书化：如"是""以""可"等常用字可写行书写法
   - 原则：不影响识别的前提下，怎么快怎么来
   - 速度提升：比楷书快30-50%

⑤ 间距均匀（整篇整齐的秘诀）
   - 字间距：不大于半个字宽（比楷书更密，节省空间）
   - 行间距：保持均匀，不忽宽忽窄
   - 段落首行缩进：两个字宽度
   - 标点：占半个字格，不单独占格（节省空间）
```

### 衡水体速成训练法（2-4周见效）

**核心方法**：不要练单字，要练"一整段话"。衡水体不是练出来的，是"写"出来的。

| 阶段 | 训练内容 | 方法 | 目标 |
|---|---|---|---|
| **第1周** | 大小控制 | 每天在方格本中抄写一段古文/课文（50字以上），只关注大小一致 | 同一行字的最大最小高度差<20% |
| **第2周** | 底部对齐 | 每天抄写一段话，关注字的底线是否在一条水平线上 | 一行字的底部波动<2mm |
| **第3周** | 加快速度 | 限时抄写：5分钟写30-50个字，在保持整齐的前提下提速 | 速度比第1周快30% |
| **第4周** | 综合实战 | 模拟考试抄写，限时+整齐+清楚三者兼顾 | 能在10分钟内写100个清晰字 |

### 衡水体快速上手技巧

```
① 握笔放松——握笔越紧字越难看，放松握笔自然就写好了
② 手腕发力——不是手指发力，手腕带动整个手写，又快又稳
③ 不必求完美——单个字写到70分即可，整体整齐才是衡水体的精髓
④ 整体优先——先保证整篇布局，再打磨单字
⑤ 笔画少写——能一笔写完的不写两笔，"口""日""目"等封闭结构可以一笔画圈

"字不是练好的，是写好的。"
—— 每天写够量，自然就练出来了
```

## 优化方案："短时高频+即时反馈"纠正法

本方案强调**短时高频、即时反馈**，每次练习都有明确的纠正目标。利用Chris的记忆力优势，通过口诀和视觉辅助强化正确姿势和笔画。

### 核心逻辑

将练字拆解为**极短的、可量化的微习惯**，每次练习都伴随**明确的纠正目标和即时反馈**。利用Chris的记忆力优势，通过口诀和视觉辅助强化正确姿势和笔画。

### 第一步：握笔与坐姿微习惯（第1周，每天2次，每次5分钟）

#### 1.1 握笔姿势口诀与检查

```
标准握笔步骤（每天练前重复，形成肌肉记忆）：
① 拇指尖与食指尖对齐，形成"OK"手势
② 把笔放在食指第三关节上（不是大拇指）
③ 拇指轻轻包住笔杆，两指尖距离笔尖约2cm
④ 笔杆靠在虎口V形处
⑤ 无名指和小指自然弯曲托住纸
⑥ 手腕放松，能左右摆动
```

**自查口诀**："拇指食指轻握笔，笔杆靠在虎口上，手腕放松能摆动"

#### 1.2 坐姿三要素

```
① 双脚平放：两脚能踩到地面（如果椅子太高，放个脚垫）
② 身体离桌一拳：卷尺量好一拳距离，贴个标记在桌上
③ 纸放正：纠正"往左歪"的关键——纸张不能向左转，纸边与桌沿平行
```

### 第二步：专项纠正微训练（第2-6周，每天2次，每次10分钟）

#### 2.1 针对"往左歪"——中线训练（每天5分钟）

**核心原理**：强化"中线意识"。利用米字格或田字格的垂直中线作为参照。

**训练内容**：

1.  **画中线**（1分钟）：在米字格中画一条竖线，作为所有字的"脊梁"。
2.  **对称字描红/书写**（3分钟）：每天选择1-2个对称字（如：中、木、王），描红或书写5遍。强调字体的中轴线与米字格中线对齐。
3.  **即时反馈**（1分钟）：每写完一个字，家长立即判断"这个字的中线和格子的中线对齐了吗？"，并用红笔标记偏差。如果偏差超过1mm，则重新书写。

#### 2.2 针对"大小不规整"——边框训练（每天3分钟）

**核心原理**：建立"边框意识"和空间感知。

**训练内容**：

1.  **方块定位**（2分钟）：在一个田字格或方格中，先画一个虚线框。Chris写的字必须填满这个框的60%-80%（不能太小也不能溢出）。
2.  **比较训练**（1分钟）：在同一行写"口""目""田""日"四个字，要求大小一致。家长即时检查并反馈。

#### 2.3 针对"笔画轻"——力度训练（每天2分钟）

**核心原理**：训练运笔力度，确保笔画清晰。

**训练内容**：

1.  **力度梯级**（1分钟）：写一横或一竖，先用力（深色），再放松（浅色），渐变练习。
2.  **数字力度练习**（1分钟）：用铅笔写数字0-9，每个写2遍，要求"铅笔颜色要能看到"。

### 第三步：速度与应用训练（第7-8周，每天2次，每次10分钟）

**核心逻辑**：练字最终要为"做题写得又快又清楚"服务。在保证正确性的前提下，逐步提升书写速度。

**训练内容**：

1.  **限时书写**（5分钟）：从之前练习过的字中随机选择5个字，限时2分钟内书写，记录"写完的字数"和"规范的字数"。
2.  **场景模拟**（5分钟）：抄写一道简单的数学题或一段英语，注重"清晰 + 合理速度"。

### 训练量总结（优化方案）

| 周次 | 每日时间 | 核心重点 | 训练形式 | 备注 |
|---|---|---|---|---|
| 第1周 | 5分钟×2次 | 握笔+坐姿矫正 | 口诀+检查 | 仅纠正姿势，不动手写字 |
| 第2-6周 | 10分钟×2次 | 中线+大小+力度 | 专项纠正 | 每次练习聚焦1-2个纠正点 |
| 第7-8周 | 10分钟×2次 | 速度+综合应用 | 限时书写+场景模拟 | 检验成果，融入实际应用 |

## 练字进度检测标准

| 阶段 | 检测内容 | 通过标准 |
|---|---|---|
| 第1周末 | 握笔姿势与坐姿 | 90%时间能保持标准姿势 |
| 第4周末 | 在米字格中写10个随机字 | ≤1个字歪斜，大小基本一致 |
| 第8周末 | 在方格本中抄写一段文字 | 清晰、可读、速度合理，无明显歪斜或大小不一 [2] |

# 目标二：学而思秘籍 9 & 10（深度理解与兴趣驱动）

## 现状分析

Chris已学至学而思秘籍第9级（2022版），当前按教材目录推进。已完成小学五年级上册内容，对科学和数学有主动兴趣。当前基线：能独立完成中等难度题，遇到新题型需要引导思路。暑假目标是从"做过"升级到"理解透彻——能讲题、能出变式题"。

## 优化方案："苏格拉底式提问+费曼输出"驱动型（推荐）

本方案强调**深度理解、主动思考和即时反馈**，充分利用Chris"好问"的特质和强大的记忆力，避免单纯的刷题 [2]。

### 核心逻辑

通过**引导式提问**激发Chris对数学概念的探索欲，结合**费曼学习法**确保其真正理解并能清晰表达解题思路。每个专题都建立"全局地图"，再进行"刻意训练" [2]。

### 每日学习安排（2小时05分钟）

> 学而思秘籍安排在上午9:15-11:30（分2节，含10分钟休息），具体流程可循环在两节课中分配。

```
① 预习与全局地图（10分钟）：
   先看本节标题和例题，尝试猜想"这个专题在讲什么？"。
   家长提供该专题的"全局地图"（包含哪些子概念、与之前学过的知识有何关联）。

② 概念讲解与苏格拉底式提问（20分钟）：
   用20/80法则抓核心概念，重点不是"做对题"，而是"理解为什么"。
   通过提问引导Chris思考："为什么这个概念是这样定义的？"、"它解决了什么问题？"。

③ 例题分析与思路拆解（15分钟）：
   学习例题，理解解题步骤。
   提问Chris："为什么这里要这样做？有没有其他方法？"，鼓励他多角度思考。

④ 独立练习与费曼输出（30分钟）：
   做3-5道练习题。**要求 Chris 将解题步骤写清楚、工整，字迹大小一致。** 也可以根据视频老师的讲解抄写关键步骤。
   每做完一道，Chris用自己的话向家长"讲一讲解题思路"（费曼法）。
   家长根据Chris的讲解判断其理解程度，而非仅仅看答案对错。

⑤ 错题分析与变式训练（15分钟）：
   如果做错，不直接改。提问："你觉得这道题哪个步骤出问题了？"，引导Chris自我纠正。
   针对错题，进行1-2道变式训练，确保概念掌握。
```

## 方案选择建议（优化版）

## 现状分析

## 优化方案："游戏化+分解训练"进阶型（推荐）

本方案在系统训练的基础上，融入更多**游戏化元素**，以保持Chris的兴趣和专注度，同时注重**技能分解与体能结合** [2]。

### 核心逻辑

将篮球技能分解为易于掌握的小动作，通过**游戏竞赛**的方式进行练习，并与体能训练相结合。强调"球感+兴趣"，而非技术完美 [2]。

### 第一阶段：球感与基础运球（第1-2周，每次60分钟）

| 训练内容 | 游戏化元素 | 目标 |
|---|---|---|
| **坐地拍球**：左右手各50次，球不超膝盖。 | "拍球比多"：看谁连续拍球次数多，记录最佳成绩。 | 强化手腕和手指力量，建立球感。 |
| **胯下拨球**：两手间来回传递，30秒×3组。 | "胯下穿梭"：规定时间内完成最多传递次数。 | 提升手眼协调和球的控制力。 |
| **绕身传球**：绕头、腰、膝各3圈，正反向。 | "球不落地"：挑战球不落地完成绕身动作。 | 提高身体与球的协调性。 |
| **原地高/低运球**：左右手各50个×3组。 | "运球不看球挑战"：尝试运球时眼睛不看球，看前方。 | 掌握基本运球姿势和节奏。 |

### 第二阶段：行进间运球与三步上篮入门（第3-4周，每次60分钟）

| 训练内容 | 游戏化元素 | 目标 |
|---|---|---|
| **直线运球**：从三分线运到篮下并返回，10次往返。 | "运球过河"：在地上画线，运球沿线走，球出界算"掉河里"。 | 提升运球稳定性，培养抬头运球习惯。 |
| **无球三步上篮脚步**：喊口令"左-右-左"或"右-左-右"。 | "节奏大师"：跟着口令，准确完成脚步动作。 | 熟悉三步上篮的节奏和步点。 |
| **持球无防守上篮**：运2步球，完成三步上篮动作。 | "上篮得分王"：记录每次上篮的得分情况（进球/碰板/未进）。 | 掌握三步上篮的完整动作流程。 |

### 第三阶段：综合运用与球场意识（第5-8周，每次60分钟）

| 训练内容 | 游戏化元素 | 目标 |
|---|---|---|
| **运球往返跑**：结合直线运球和变速。 | "闪电运球"：规定时间内完成往返跑，挑战最快速度。 | 提升运球速度和变向能力。 |
| **三步上篮实战**：从不同角度进行上篮。 | "角度挑战"：尝试从左侧、右侧、中间进行上篮。 | 提高上篮的成功率和适应性。 |
| **投篮练习**：罚球线投篮10次。 | "神射手"：记录投篮命中率，争取更高分数。 | 巩固投篮技术。 |
| **半场模拟**：运球+传球+上篮，模拟比赛场景。 | "迷你比赛"：与家长进行简单的1对1或2对2，培养球场意识。 | 提升综合技能和团队协作意识。 |

# 目标四：英语发音与口语表达（从"背过"到"地道"）

## 现状分析

Chris英语基础极佳（蓝思900，NC2/3全文背诵，NC4前半背诵），发音短板在：th音（/θ//ð/）、v/w区分、长元音短元音区分、句子的连读弱读。节奏感偏"一字一顿"，缺乏自然的语调起伏。当前基线：能读复杂文本但发音不地道，有中式口音痕迹。

## 优化方案："影子跟读+音素纠正+主题演讲"进阶型（推荐）

本方案利用Chris强大的记忆力，通过**影子跟读**纠正发音节奏，结合**音素专项训练**解决口音问题，并通过**主题演讲**提升口语表达能力 [2]。

### 核心逻辑

从模仿到内化，再到自主表达。先通过高强度模仿建立正确的语音语调，再针对性纠正音素，最终通过结构化训练提升口语输出能力 [2]。

### 第一阶段：听感与节奏建立（第1-2周，晚上45分钟）

```
材料：新概念英语3 课文音频（Chris已背过，可专注发音）

步骤：
① 盲听与节奏感知（5分钟）：不看书，只听音频，注意语速、语调、重音和连读。
② 逐句影子跟读（15分钟）：播放一句，Chris紧随其后模仿读一句。重点模仿"语调升降"和"连读弱读"。
③ 对比检查与自我纠正（10分钟）：Chris自己读一句，再听原音频对比。提问："我读的和原音哪里不同？"，引导自我发现。
④ 录音分析（15分钟）：录下Chris跟读的片段，与原音频进行对比分析，找出主要问题。
```

**本阶段检测**：录一段Chris朗读课文的声音，与原音频对比，看其语调、节奏是否有所改善。

### 第二阶段：音素纠正与语流优化（第3-4周，晚上45分钟）

```
每天针对2-3个音素做专项训练，结合语流中的连读、弱读、爆破等现象。

常见发音问题（中国儿童常见）[4]：
① /θ/ 和 /ð/（th 音） — 用牙齿轻咬舌尖，感受气流。
② /v/ — 上齿轻咬下唇，感受震动。
③ 长元音 vs 短元音（如 ship vs sheep）— 区分发音时长。
④ /l/ 和 /n/ 区别 — 感受舌头位置。

训练步骤：
① 音素示范与口型观察（5分钟）：听标准发音，观察发音口型（可借助镜子或视频）。
② 单词与短语练习（15分钟）：练习包含目标音素的单词和短语，重复发音。
③ 语流练习（15分钟）：在句子中练习连读、弱读、爆破等现象，如"an apple"的连读。
④ 绕口令/韵律诗（10分钟）：选择包含目标音素的绕口令或韵律诗进行朗读，提升发音流畅度。
```

### 第三阶段：主题演讲与自由表达（第5-8周，晚上45分钟）

**核心逻辑**：将所学的发音技巧应用于实际口语输出，提升Chris用英语完整表达想法的能力 [2]。

```
每周选择1-2个Chris感兴趣的主题（如太空、恐龙、马斯克等）进行准备。
```

# 目标八：游泳（自由泳技术精进与耐力提升）

## 现状

## 优化方案："分解纠正+游戏挑战"进阶型（推荐）

本方案在原有分解训练的基础上，增加**游戏挑战**和**可视化反馈**，提升Chris学习游泳的兴趣和效率 [2]。

### 核心逻辑

将自由泳技术分解为打腿、划水、呼吸、配合四大要素，通过针对性练习和游戏化挑战，逐步纠正动作，提升耐力 [2]。

### 自由泳动作纠正（四大要点与游戏化）

| 动作要点 | 常见问题 | 纠正方法与游戏化 |
|---|---|---|
| **打腿** | 膝盖弯曲过多，大腿不动小腿动。 | **纠正**：趴在池边或浮板上，大腿带动小腿打水，脚踝放松，脚尖指向池底。\n**游戏化**："水花挑战"——打腿时水花要小而密，水花越大扣分。 |
| **划水** | 手肘过低，划水效率低。 | **纠正**："高肘抓水"——手入水后，先转肘再转手，想象"用手抱一个球"。\n**游戏化**："划水力量计"——用手划水时，感受水流的阻力，力量越大得分越高。 |
| **呼吸** | 抬头呼吸，身体扭曲。 | **纠正**："划水时转头"——不是抬头，而是随身体滚动转头。\n**游戏化**："呼吸平衡木"——呼吸时身体要保持平衡，不能左右摇晃。 |
| **完整配合** | 手脚不协调，呼吸时动作打断。 | **纠正**：2次划水+1次呼吸的节奏。\n**游戏化**："节奏大师"——在水中默念"划-划-呼"的节奏，保持动作连贯。 |

### 8周游泳计划

| 周次 | 训练重点 | 练习内容（融入游戏化） |
|---|---|---|
| 第1-2周 | 打腿+划水分解 | 扶板打腿25米×5组（水花挑战），单臂划水×10次/臂（划水力量计）。 |
| 第3-4周 | 呼吸纠正 | 侧身打腿+呼吸×10次/侧（呼吸平衡木），完整配合2次划水+1次呼吸（节奏大师）。 |
| 第5-6周 | 完整配合 | 25米5组（完整自由泳，记录每组的呼吸次数，挑战最少呼吸次数）。 |
| 第7-8周 | 综合+耐力 | 50米自由泳（不计时，挑战最长距离），25米×5组（计时，挑战最快速度）。 |

> 本体系将整个暑假计划转化为每日微信群可执行的清单，包含任务编号、评估方法、反馈闭环和动态调整机制。

## 一、核心流程

```
每日早7:00 → 在微信群发布当天计划（带编号）
        → Chris执行各任务
        → 每项任务完成后，按指定方法验证
        → 家长在群里回复编号 + ✅/评分
        → 晚21:00 收集当日反馈 → 调整次日计划
        → 周末生成周总结 → 调整下周计划
```

**微信群成员**：爸爸、妈妈、Chris（可看）、其他家庭成员

## 二、任务编号系统

### 编号格式

```
D{day}-{seq}
```

- `D1` = 暑假第1天（2026/7/6），`D2` = 第2天，以此类推
- `seq` = 两位顺序号（01-20）

### 编号与科目对照

| 编号范围 | 科目 | 评估方法 |
|---|---|---|
| DxC1~2 | 🏀 篮球/🏓 乒乓球 | 家长观察评分 |
| DxC3~4 | 📚 学而思秘籍 | 拍照 → Chris-数学-考试官 |
| DxC5 | ✍️ 练字（第1次·工整） | 拍照 → Chris-练字-考试官 |
| DxC6 | 📖 Reading Explorer | 费曼提问+英语复述 |
| DxC7 | 📖 古文背诵（新篇） | 语音 → Chris-古文-考试官 |
| DxC8 | 📘 新概念英语3复习 | 语音 → Chris-新概念背诵-考试官 |
| DxC9 | ✍️ 练字（第2次·工整） | 拍照 → Chris-练字-考试官 |
| DxC10 | 🗣️ 英语发音/口语 | 录音 → Chris-英语发音-考试官 |

## 四、各科目评估方法细则

### 4.1 📖 古文背诵 — 费曼法+默写

| 步骤 | 方法 | 通过标准 |
|---|---|---|
| ① 复述 | Chris用自己的话讲一遍古文大意 | 能准确说出核心意思 |
| ② 费曼提问 | 家长问一个引导性问题（见每日费曼提问） | Chris能清晰回答 |
| ③ 关键句 | 背诵本篇最核心的一句 | 一字不差 |
| ④ 默写（隔天） | 次日复习时默写本篇 | 正确率≥90% |

**微信群反馈示例**：
```
D1C6 ✅ 5分 能完整讲出咏鹅画面，关键句一字不差
```

### 4.2 📚 学而思秘籍 — 拍照+豆包评估

| 步骤 | 方法 |
|---|---|
| ① 做题 | Chris独立完成练习题 |
| ② 拍照 | 家长拍下解题步骤，发到群里 @豆包评估 |
| ③ 豆包评估 | 将照片发给豆包数学评估智能体（见下方配置） |
| ④ 结果回复 | 豆包评估后家长在群里回复评分 |

**微信群反馈示例**：
```
D1C2 ✅ 4分 豆包评估：概念理解5/5，步骤完整4/5，计算准确3/5→需加强进位
```

### 4.3 ✍️ 练字 — 拍照+豆包评估

| 步骤 | 方法 |
|---|---|
| ① 书写 | Chris在米字格中完成练习 |
| ② 拍照 | 家长拍下练字成果 |
| ③ 豆包评估 | 将照片发给豆包写字评估智能体 |
| ④ 结果回复 | 获取豆包评分和建议后回复 |

**微信群反馈示例**：
```
D1C4 ✅ 3分 豆包评估：中线4/5，大小3/5，力度2/5→建议加强手腕放松
```

**练字指导思想**：专项练字时间只练**工整**（用米字格、中线游戏）；古文默写和数学写步骤时自然练**快速**（要求字迹可辨认即可）

### 4.4 🗣️ 英语发音 — 录音+豆包评估

| 步骤 | 方法 |
|---|---|
| ① 练习 | Chris完成影子跟读/音素练习 |
| ② 录音 | 用微信语音消息录一段Chris朗读 |
| ③ 豆包评估 | 将语音发给豆包英语发音评估智能体 |
| ④ 结果回复 | 获取评分和建议后回复 |

**微信群反馈示例**：
```
D1C10 ✅ 4分 豆包评估：口音4/5，节奏3/5，流利度4/5→连读需加强
```

### 4.5 📖 Reading Explorer — 费曼提问+英语复述

| 步骤 | 方法 | 通过标准 |
|---|---|---|
| ① 词汇 | Chris用5个核心词各造一句 | 造句合理 |
| ② 主旨 | 用一句话概括文章主旨 | 抓住核心 |
| ③ 费曼英语复述 | Chris用英语讲一遍文章大意 | 3句话以上，逻辑连贯 |
| ④ 深度提问 | 家长问1个why问题 | Chris能给出自己的看法 |

**微信群反馈示例**：
```
D1C5 ✅ 5分 英语复述流畅，能说出科学家选择这个方法的原因
```

### 4.6 📘 NC3 — 背诵检测+发音

| 步骤 | 方法 | 通过标准 |
|---|---|---|
| ① 背诵 | Chris背诵课文片段 | 流畅无卡顿 |
| ② 语法 | 指出课文中1个语法点，自己举例 | 例子正确 |
| ③ 影子跟读 | 跟读1段，重点纠正发音 | 语调明显改善 |

**微信群反馈示例**：
```
D1C7 ✅ 4分 背诵流畅，语法点举例正确，/θ/音还需练
```

### 4.7 🏀 / 🏓 运动 — 家长观察

| 项目 | 观察重点 | 评分维度 |
|---|---|---|
| 篮球 | 运球不低头？上篮步法？ | 技能/态度/进步 |
| 乒乓球 | 球感？正手动作框架？ | 球感/兴趣 |
| 游泳 | 打腿？换气？ | 动作/耐力 |

**微信群反馈示例**：
```
D1C1 ✅ 4分 拍球稳定，胯下拨球30s完成3组
```

## 七、动态调整规则

| 情况 | 调整方式 |
|---|---|
| **当日某任务未完成** | 次日计划中标记为`D{N+1}-R** 🔄补：D{N}C**`，优先补完 |
| **连续2天某任务评分<3分** | 降低该任务难度/减少量，或重新讲解概念 |
| **连续3天某任务评分≥5分** | 加快进度或增加难度 |
| **学而思章节与实际情况不符** | 在微信群中@修改计划，调整后续几天的编号 |
| **Chris状态不好** | 减少当天30%任务量，优先保证核心科目（练字+古文） |
| **外出/远足日** | 当天只保留练字+古文+NC3三项，其余取消 |
| **计划临时有变** | 家长在群里发`📋 计划更新：D{N}C**改为...` |

### 记忆曲线复习节奏嵌入

每天的 `D{N}CR01` 和 `D{N}CR02` 按以下规则自动生成：

| 复习时间点 | 复习内容 | 对应编号 |
|---|---|---|
| **次日** | 回顾前1天的古文和RE | D{N}CR01 ← D{N-1}C06, D{N}CR02 ← D{N-1}C05 |
| **2天后** | 回顾2天前的古文 | D{N}CR03 ← D{N-2}-06 |
| **4天后** | 回顾4天前的古文 | D{N}CR04 ← D{N-4}-06 |
| **7天后（复习日）** | 回顾本周全部古文 | 周复习日全部回顾 |
