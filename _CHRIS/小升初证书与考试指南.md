## 二、各考试详细说明

### 2.1 小托福（TOEFL Junior）

> **优先级最高**：三公申请中，小托福850+几乎是标配，比KET/PET含金量更高。

#### 考试信息
| 项目 | 内容 |
|---|---|
| **全称** | TOEFL Junior（托福青少年版） |
| **组织者** | ETS（美国教育考试服务中心） |
| **中国区运营** | ETS中国 / 教育部考试中心 |
| **考试内容** | 听力（42题）、语言形式与含义（42题）、阅读理解（42题） |
| **考试时长** | 约2小时 |
| **满分** | 900分（每部分150分，共3部分） |
| **目标分数** | **850+**（三公门槛级别） |
| **成绩有效期** | 2年 |
| **适合年龄** | 11-17岁（但实际上海7-8岁高分考生很多） |

#### 考试时间与报名
| 项目 | 内容 |
|---|---|
| **考试频次** | 每月都有（全国多个城市设考点） |
| **报名方式** | 官网 `www.toefljuniorchina.com` 在线报名 |
| **报名费用** | 约600元/次 |
| **上海考点** | 上海多个国际学校、大学设有考点 |
| **出分时间** | 考后约2周 |

#### 分数对标
| 分数 | 对应CEFR等级 | 三公竞争力 |
|---|---|---|
| 850+ | B2（中高级） | ⭐⭐⭐⭐⭐ 非常强 |
| 800-845 | B1（中级） | ⭐⭐⭐⭐ 有竞争力 |
| 750-795 | A2（中初级） | ⭐⭐⭐ 基础水平 |
| <750 | A1（初级） | ⭐⭐ 需要加强 |

#### Chris 的备考策略
| 年级 | 目标分数 | 准备方式 |
|---|---|---|
| 二年级 | 模拟测试摸底 | Reading Explorer 阅读量积累 |
| 三年级上 | 750+ | 开始系统备考+真题练习 |
| 三年级下 | 800+ | 目标达成 |
| 四年级上 | 850+ | 最终目标分数 |

### 2.2 AMC8（American Mathematics Competition 8）

> **数学能力核心证明**：三公和顶尖初中非常看重AMC成绩，AMC8前5%或前1%是数学的有力证明。

#### 考试信息
| 项目 | 内容 |
|---|---|
| **全称** | American Mathematics Competition 8 |
| **组织者** | MAA（美国数学协会） |
| **中国区运营** | ASDAN 阿思丹（中国独家合作方） |
| **考试内容** | 25道选择题，涵盖数论、几何、代数、组合、概率 |
| **考试时长** | 40分钟 |
| **满分** | 25分 |
| **奖项层级** | 见下表 |
| **成绩有效期** | 无有效期（比赛成绩） |
| **适合年龄** | 8年级及以下（上海通常3-5年级参加） |

#### 奖项与目标
| 奖项/分数 | 条件 | 三公竞争力 |
|---|---|---|
| **全球前1%（Honor Roll of Distinction）** | 21-22分以上（每年浮动） | ⭐⭐⭐⭐⭐ 极强 |
| **全球前5%（Honor Roll）** | **17分左右** | ⭐⭐⭐⭐ 目标线 |
| **全球前15%（Achievement Roll）** | 13分左右 | ⭐⭐⭐ 基础水平 |
| **学校优秀奖** | 校内最高分 | ⭐⭐ 辅助 |

#### 考试时间
| 项目 | 内容 |
|---|---|
| **考试时间** | 每年1月（通常1月中旬） |
| **报名截止** | 前一年12月中旬 |
| **报名方式** | 通过ASDAN阿思丹报名（学校或机构） |
| **报名费用** | 约600-800元/次 |
| **出分时间** | 考后约4-6周 |

#### 考试形式
| 形式 | 内容 |
|---|---|
| **线下纸笔考试** | 普通模式 |
| **在线考试** | 部分支持 |
| **语言** | 中英文双语试卷 |

#### Chris 的备考策略
| 年级 | 目标 | 准备方式 |
|---|---|---|
| 二-三年级 | 打基础 | 学而思秘籍 + 浅奥 |
| 三年级 | 首次考 → 10-13分 | AMC8真题训练 |
| 四年级 | **17+分（前5%）** | 系统AMC8刷题+专题突破 |

### 2.5 KET / PET（剑桥通用英语）

> 小托福的替代品。如果小托福报名不便或需要更多考试机会，KET/PET也是好的选择。

| 考试 | 对标CEFR | 对应小托福分数 | 适用年级 |
|---|---|---|---|
| KET（Key English Test） | A2 | 小托福约750 | 二-三年级 |
| PET（Preliminary English Test） | B1 | 小托福约800-845 | 三-四年级 |
| FCE（First Certificate） | B2 | 小托福约850+ | 五年级+ |

#### KET/PET 考试信息
| 项目 | 内容 |
|---|---|
| **组织者** | 剑桥大学英语考评部 |
| **中国区运营** | 教育部考试中心 |
| **考试频次** | 每月都有 |
| **报名方式** | 官网报名或通过机构 |
| **报名费用** | KET约500元，PET约600元 |
| **考试内容** | 阅读写作（60min）、听力（30min）、口语（8-10min） |

## 五、关键提醒

1. **小托福是三公的"硬门槛"**：据历年家长反馈，三公面谈的简历筛选中，小托福850+几乎是标配
2. **AMC8 前5%（约17分）是数学能力的证明**：虽然上海没有强制要求AMC，但有三公录取的考生基本上都有AMC成绩
3. **证书不是万能**：三公面谈除了看证书，还有现场的表现（英语面谈、数学逻辑、百科）
4. **Chris 的优势是英语**：英语是他最大的杠杆，小托福850+应该比同龄人更容易达到
5. **数学要抓紧**：Chris 目前在学而思秘籍9-10（五年级），但奥数竞赛（AMC8）考的是系统性思维，不是超前学就能拿高分的，需要专门的训练
