### 2.4 神秘考（SMK）详解

> **神秘考是进入华育理科班的核心渠道**，也是上海顶尖初中选拔的主要方式。

#### 神秘考的特点
| 特点 | 说明 |
|---|---|
| 非公开 | 不公开发布信息，仅通过特定机构定向通知 |
| 时间 | 五年级上学期10月-12月 |
| 报名方式 | 不对外报名，机构筛选后通知 |
| 考试内容 | 数学（核心）+ 英语 + 语文 |
| 数学难度 | 约等于AMC8 前5%水平 + 小奥拓展 |
| 通知方式 | 通过机构或家长群通知 |

#### 如何获得神秘考资格

| 路径 | 要求 | 难度 |
|---|---|---|
| **学而思创新班/集训队** | 三年级起进入学而思创新班或集训队 | ⭐⭐⭐⭐⭐ |
| **四季金牌班** | 三年级起进入四季教育金牌班 | ⭐⭐⭐⭐ |
| **蘑菇培优卓越班** | 三年级起进入蘑菇卓越班 | ⭐⭐⭐⭐ |
| **其他机构高端班** | 乐读、新舟等 | ⭐⭐⭐ |
| **小学推荐** | 部分知名小学有推荐名额（如世外、逸夫、盛大等） | ⭐⭐⭐⭐⭐ |

#### 神秘考考试内容（根据历年家长反馈）

| 科目 | 题量 | 难度 | 考查重点 |
|---|---|---|---|
| 数学 | 25-30题 | 高 | 奥数专题：数论、几何、组合、行程、应用题 |
| 英语 | 阅读+语法 | 中高 | 相当于小托福800+水平 |
| 语文 | 阅读+古文 | 中 | 古诗文、阅读理解 |

### 2.5 华育历年面谈/考试经验

> 以下为历年家长反馈的经验总结，供参考。

**数学部分**：
- 计算量大，要求速度快
- 数论和几何是重点，占分比例高
- 部分题目达到AMC8 20分难度
- 近年开始有"数学思维"类型题目（非计算，而是逻辑推理）

**英语部分**：
- 阅读篇幅长，约400-500词
- 词汇量要求大（相当于小托福水平）
- 有完形填空和语法题

**语文部分**：
- 古诗文默写和理解
- 阅读理解和作文
- 需要一定的文学素养积累

**面谈部分**：
- 英语面谈：自我介绍+话题讨论
- 数学面谈：口头解题
- 行为观察：礼貌、专注度、表达能力

## 第八章：核心原则

> 已写入 Claude 记忆：

**用最短的时间来实现最确定的目标**

每个学习决策前问：
1. 这条路径是不是花时间最少的？
2. 结果是不是最确定的？
3. 有没有更快的替代方案？

在这个原则下，Chris 的优先级：
1. **小托福850+** → 确定性最高（英语是Chris优势）
2. **AMC8 前5%** → 确定性高（数学有基础，且路径清晰）
3. **神秘考准备** → 需要机构高端班通道
4. **英语面谈** → 需要长期积累，暑假就在练
5. **USACO/物理竞赛** → 长期规划，不急在一时
