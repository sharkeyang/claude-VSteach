# 目标一：练字（暑假最大任务）

## 现状分析

| 问题 | 可能的技术原因 | 严重程度 |
|---|---|---|
| 字整体向左歪 | ① 握笔时食指弯曲过度或拇指过于靠上 ② 纸张向左侧倾斜过度 ③ 书写时手腕内扣 | ⭐⭐⭐ |
| 字体大小不一 | ① 缺乏"边框意识"——没感知字在格子中的空间 ② 运笔控制力弱 | ⭐⭐ |
| 笔画轻 | ① 握笔过紧导致手指僵硬，运笔受限 ② 力量训练不足 | ⭐ |

---

## 方案A：密集纠正型（推荐——针对 Chris 多次纠正无改善的情况）

> Chris 之前多次纠正无明显改善，说明需要"高密度 + 强反馈"的方法。

### 核心逻辑
Chris 记忆力强、好动，所以不适合"长时间练"而适合"短时间高密度纠正"。每天 20 分钟，但这 20 分钟内要高度专注、有即时反馈。

### 第一步：握笔与坐姿彻底矫正（第1周，每天10分钟）

#### 1.1 握笔姿势（图片编号法）
Chris 记性好，用"编号法"让他记住每个手指的位置：

```
标准握笔步骤（每天练前重复）：
① 拇指尖与食指尖对齐，形成"OK"手势
② 把笔放在食指第三关节上（不是大拇指）
③ 拇指轻轻包住笔杆，两指尖距离笔尖约2cm
④ 笔杆靠在虎口V形处
⑤ 无名指和小指自然弯曲托住纸
⑥ 手腕放松，能左右摆动
```

**自查口诀**："拇指食指轻握笔，笔杆靠在虎口上，手腕放松能摆动"

#### 1.2 坐姿三要素
```
① 双脚平放：两脚能踩到地面（如果椅子太高，放个脚垫）
② 身体离桌一拳：卷尺量好一拳距离，贴个标记在桌上
③ 纸放正：纠正"往左歪"的关键——纸张不能向左转，纸边与桌沿平行
```

### 第二步：专项训练（第2-6周，每天20分钟）

#### 2.1 针对"往左歪"——中线训练法（每天10分钟）

**核心原理**：字往左歪是因为缺乏"中线意识"。米字格或田字格能提供垂直中线的参考。

**训练内容**：
```
① 画中线（2分钟）：在米字格中画一条竖线，作为所有字的"脊梁"
② 写对称字（5分钟）：每天写3个对称字，每个写10遍
   第2周：中、木、来、十、王
   第3周：中、平、本、果、井
   第4周：林、森、杏、森、架
   第5周：走、道、题、超、赶
   第6周：综合复习
③ 自主检查（3分钟）：每写完一个字，自己判断"这个字的中线和格子的中线对齐了吗"
```

**针对"向左歪"的专项技巧**：
```
每次写字前，先用铅笔在格子里画一条淡淡的竖线（中线）。
写每一个字时，想象有一个"垂直的绳子"从上面穿过字的中间。
写完后，拿尺子量一下：字的中心线和格子中心线的偏差。
如果偏差超过1mm，就重新写这个字。
```

#### 2.2 针对"大小不规整"——边框感知训练（每天5分钟）

```
① 方块定位（3分钟）：
   在一个田字格或方格中，先画一个框（左上、左下、右上、右下四个角）
   写的字必须填满这个框的60%-80%（不能太小也不能溢出）
   
② 比较训练（2分钟）：
   在同一行写"口""目""田""日"四个字
   这四个字的大小应该一样大——如果不一样，说明空间感知需要训练
```

#### 2.3 针对"笔画轻"——运笔力度训练（每天5分钟）

```
① 力度梯级（3分钟）：
   写一横：先用力（深色），再放松（浅色），渐变练习
   写一竖：同样操作
   
② 数字力度练习（2分钟）：
   用铅笔写数字0-9，每个写3遍，要求"铅笔颜色要能看到"
```

### 第三步：速度训练（第7-8周，每天20分钟）

> 练字最终要为"做题写得又快又清楚"服务。

```
① 限时训练（10分钟）：
   5分钟内写10个字（从之前练过的字中随机选）
   记录"写完的字数"和"规范的字数"
   
② 数字书写（5分钟）：
   0-9每个写5遍，要求"5秒内写一个清晰的数字"
   这是数学考试的基础
   
③ 场景模拟（5分钟）：
   抄写一道简单的数学题或一段英语
   注重"清晰 + 合理速度"
```

### 训练量总结（方案A）

| 周次 | 每日时间 | 核心重点 | 累计字数 |
|---|---|---|---|
| 第1周 | 10分钟×1次 | 握笔+坐姿矫正 | 仅纠正不动手 |
| 第2周 | 30分钟×2次 | 中线训练+对称字（3个×10遍） | 30字×2/天 |
| 第3周 | 30分钟×2次 | 中线+对称字（3个×10遍）+力度 | 30字×2/天 |
| 第4周 | 30分钟×2次 | 中线+对称字+大小控制 | 30字×2/天 |
| 第5周 | 30分钟×2次 | 综合训练 | 30字×2/天 |
| 第6周 | 30分钟×2次 | 综合训练+检测 | 30字×2/天 |
| 第7周 | 30分钟×2次 | 速度训练 | 50字×2/天 |
| 第8周 | 30分钟×2次 | 速度+综合检测 | 50字×2/天 |

```

④ 独立练习（15分钟）：
   做3-5道练习题
   每做完一道，用费曼法：讲一讲解题思路

⑤ 错题分析（5分钟）：
   如果做错，不直接改
   问："你觉得这道题哪个步骤出问题了？"
```

# 目标三：篮球（户外场地5分钟可达）

## 现状分析

Chris 篮下投篮10进5-7个，说明基本的投掷能力不错。需要加强的是：
1. 运球能力（现在是否会运球？）
2. 三步上篮（完全没学过还是学过不熟练？）
3. 球场意识

---

## 方案A：系统训练型（推荐——适合暑假集中提升）

### 核心逻辑
7岁儿童篮球训练的核心不是"技术完美"，而是"球感+兴趣"。Chris 好动，篮球正好是释放精力的出口。

### 第一阶段：球感建立（第1-2周，每次15分钟）

```
动作① 坐地拍球：
   坐在地上，用右手拍球50次，左手拍球50次。
   目标：球弹起高度不超过膝盖。
   技巧：掌心空出，用手指拍球，手腕发力。

动作② 胯下拨球：
   两脚分开，球从胯下在两手间来回传递。
   每组30秒，做3组。

动作③ 绕身传球：
   球绕头部→腰部→膝盖各3圈，然后反向。
   每组3圈，做3组。

动作④ 投篮巩固（已有基础）：
   篮下投篮，10球×5组。
   本周重点：记录每组的进球数。
```

### 第二阶段：运球入门（第3-4周，每次20分钟）

```
动作① 原地高运球（5分钟）：
   两脚分开与肩同宽，右手运球到胸部高度。
   50个×3组，左右手各。
   目标：球不弹到脚上。

动作② 原地低运球（5分钟）：
   两脚分开，右手运球到膝盖高度。
   50个×3组，左右手各。
   目标：手指发力，球速均匀。

动作③ 直线运球（10分钟）：
   从三分线运球到篮下（约5-7米），再运回来。
   10次往返。
   目标：运球时眼睛不看球，看前方。
```

### 第三阶段：三步上篮（第5-6周，每次20分钟）

```
动作① 无球脚步练习（5分钟）：
   站在原地，喊"左—右—左"或"右—左—右"的节奏
   （右手上篮：先跨左脚，再跨右脚，最后左脚起跳）
   
动作② 持球无防守练习（10分钟）：
   从罚球线开始，运2步球，然后：
   第1步（右）：跨右脚，收球
   第2步（左）：跨左脚，双手举球到右肩上方
   第3步（右）：起跳，右手将球轻轻碰板上篮

动作③ 完整上篮（5分钟）：
   从三分线运球至篮下，完成三步上篮。
   10次，记录进球数。
```

### 第四阶段：综合训练（第7-8周，每次20分钟）

```
① 运球往返跑（5分钟）
② 三步上篮10次（5分钟）
③ 投篮练习：罚球线投篮10次（5分钟）
④ 半场模拟（5分钟）：运球+传球+上篮
```

# 目标四：英语发音与口语表达

## 现状分析

Chris 英语优势巨大（蓝思900、NC3背诵基础），但发音和口语表达是短板。

**具体问题**：
1. 口音有偏差——可能某些音素发音不标准
2. 节奏感弱——读英文时"一字一顿"，缺乏连读、弱读、重音
3. 英语表达/演讲——缺乏"用英语完整表达想法"的训练

---

## 方案A：影子跟读法（推荐——利用 Chris 英语基础）

### 核心原理
影子跟读（Shadowing）是公认的第二语言发音训练方法。Chris 记忆力强、有大量背诵基础，最适合这种方法。

### 第一阶段：听感建立（第1-2周，晚上45分钟）

```
材料：新概念英语3 课文（Chris 已经背过，可以专注在发音上）

步骤：
① 盲听（3分钟）：不看书，只听音频，注意发音节奏
② 逐句影子跟读（7分钟）：
   播放一句，暂停，Chris 模仿读一句
   重点：模仿"语调升降"而不是只模仿音节
③ 对比检查（5分钟）：
   自己读一句，听音频再读一句
   找出"我读的 和 原音哪里不同"
```

**本周检测**：录一段 Chris 读课文的声音，与原音频对比。

### 第二阶段：音素纠正（第3-4周，晚上45分钟）

```
每天针对2-3个音素做专项训练：

常见发音问题（中国儿童常见）：
① /θ/ 和 /ð/（th 音） — 用牙齿轻咬舌尖
② /v/ — 上齿咬下唇
③ 长元音 vs 短元音（ship vs sheep）
④ /l/ 和 /n/ 区别

训练步骤：
① 听一遍标准发音（2分钟）
② 看发音嘴形（视频或镜子）（3分钟）
③ 读含该音的所有单词（5分钟）
④ 在句子中读该音（5分钟）
```

### 第三阶段：节奏训练（第5-6周，晚上45分钟）

```
重点：连读、弱读、重音

① 连读训练（5分钟）：
   例："Not at all" → No-ta-tall
   "I am" → I'm
   "What is it?" → Who-ti-zit?
   
   用 Chris 已背的新概念课文提取连读例子

② 弱读训练（5分钟）：
   冠词（a, an, the）、介词（to, of, for）在句子中读弱化版本
   The book → /ðə buk/ 而不是 /ðiː buk/
   
③ 句子重音（5分钟）：
   一个句子中，只有核心词重读
   "I **love** **reading** this **book**"
```

# 目标八：游泳（自由泳巩固）

## 现状
Chris 已学会自由泳简单动作但不熟练。每周可以带去游泳2次。

## 自由泳动作纠正（四大要点）

```
① 打腿：
   常见问题：打腿时膝盖弯曲太多，大腿不动小腿动
   纠正方法：
   - 趴在池边或浮板上，大腿带动小腿打水
   - 要求：脚踝放松，脚尖要指向池底
   - 练习：扶板打腿25米×5组

② 划水：
   常见问题：划水时手肘过低，划水效率低
   纠正方法：
   - "高肘抓水"——手入水后，先转肘再转手
   - 想象"用手抱一个球"
   - 练习：单臂划水×10次/每臂

③ 呼吸：
   常见问题：抬头呼吸、呼吸时身体扭曲
   纠正方法：
   - "划水时转头"——不是抬头，而是随身体滚动转头
   - 练习：侧身打腿+呼吸×10次/每侧

④ 完整配合：
   常见问题：手脚不协调，呼吸时动作打断
   纠正方法：
   - 2次划水+1次呼吸的节奏
   - 练习：慢速游25米×5组，每次呼吸都要稳定
```

## 8周游泳计划

| 周次 | 训练重点 | 练习内容 |
|---|---|---|
| 第1-2周 | 打腿+划水分解 | 扶板打腿25米×5组，单臂划水×10次/臂 |
| 第3-4周 | 呼吸纠正 | 侧身打腿+呼吸×10次/侧，完整配合2次+1呼吸 |
| 第5-6周 | 完整配合 | 25米5组（完整自由泳），记录每组的呼吸次数 |
| 第7-8周 | 综合+耐力 | 50米自由泳（不计时），25米×5组（计时） |
