第 9 课 · 高级 Prompt 模式

掌握了基础 Prompt 后，这些高级技巧能让你从 Claude 那里获得更精准、更深入的答案。

1. 思维链（Chain of Thought）

让 Claude 把思考过程说出来，而不是直接给答案。这对复杂问题特别有效。

🔍 示例：

「这道题：小明有 15 个苹果，给了小红 5 个，又买了 3 个，现在有多少？请先一步步分析再给出答案。」

2. 角色扮演（Role Prompting）

给 Claude 一个身份，它会以这个身份的知识体系和表达方式来回答。

🎭 示例：

「你是一位有 20 年经验的基金经理，请用通俗易懂的方式解释什么是量化交易。」

3. 格式化输出（Structured Output）

明确指定输出的格式，让结果可以直接使用。

📋 示例：

「请用 JSON 格式输出，包含：股票代码、名称、当前价格、涨跌幅。」

4. 对比分析（Comparative Analysis）

同时给出多个选项，让 Claude 做对比分析。

⚖️ 示例：

「请对比 Python 和 Go 在以下维度：开发效率、运行性能、学习曲线、生态系统。用表格展示。」

5. 迭代优化（Iterative Refinement）

先让 Claude 出一个版本，然后逐步提出改进要求。这是最实用的模式。

🔄 流程：

第一轮：「写一份 Python 脚本读取 CSV 文件。」
第二轮：「加个进度条显示处理进度。」
第三轮：「改成异步处理，同时处理 5 个文件。」

💡 核心原则： 这些模式可以组合使用。 比如"角色扮演 + 思维链 + 格式化输出"——"你是一位资深数据分析师，请一步步分析这份销售数据，输出为 Markdown 表格。"
