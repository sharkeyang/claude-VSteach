第 5 课 · Artifacts 与预览

Artifacts 是 Claude 的一项独特功能——它不仅能和你聊天，还能在对话中生成可交互的内容，直接在浏览器里预览。

什么是 Artifacts？

当 Claude 生成了代码、文档、图表等内容时，它可以把这些内容显示在对话旁边的独立窗口中，你可以直接查看效果、复制代码、下载文件。

Artifacts 能做什么？

  • HTML 页面预览 — 让 Claude 写一个网页，在 Artifacts 窗口里直接看到效果
  • SVG 图表 — 生成流程图、思维导图、K 线图等可视化内容
  • React 组件 — 实时预览交互式 UI 组件
  • Markdown 文档 — 生成格式化文档，即时查看效果
  • 代码片段 — 带语法高亮的代码，方便复制
💡 SHARK 实战场景：
  • 让 Claude 生成一个 HTML 教学页面 → 直接在 Artifacts 里预览效果
  • 生成一个 SVG 缠论结构图 → 预览后下载
  • 写一个 Python 脚本 → Artifacts 里查看完整代码再复制保存

使用技巧

  • 明确要求 Artifacts — 说"请在 Artifacts 中生成"会触发独立窗口
  • 迭代修改 — 对 Artifacts 内容不满意，直接说"改大标题"或"换个颜色"
  • 下载保存 — Artifacts 支持下载为文件，方便保存到本地
  • 不适合的场景 — 非常长的代码或超大型文件，Artifacts 有大小限制
