第 1 课 · 为什么 Claude 与众不同

Claude 是谁？

Claude 是 Anthropic 公司开发的 AI 助手，擅长对话、分析、写作和编程。它像一位理解力极强的伙伴，而不是一个只会执行指令的工具。

Claude 的核心优势

  • 超长上下文 — 可以一次性处理数百页文档、整个代码库，不会"忘记"前面聊过什么
  • 理解细腻 — 能够捕捉语气、意图和隐含需求，不只是匹配关键词
  • 安全可靠 — 经过大量对齐训练，不容易偏离指令
  • 多模态 — 支持文字、图片、文件上传，理解和生成内容
  • 可扩展 — 通过 MCP 和 Skills 连接外部工具，不局限于对话
💡 理解上下文窗口： Claude 的上下文窗口就像它的"工作台"——上面能同时放多少信息决定它做多复杂的事。 放一本 500 页的书，它也能读进去。 但这个空间是有限的，聊久了旧信息会被"挤出去"，所以关键信息最好放在项目知识或 CLAUDE.md 中。

适用场景一览

  • 编程 — 写代码、Debug、重构、代码审查
  • 写作 — 文章、报告、邮件、文档
  • 分析 — 数据解读、文档总结、趋势判断
  • 学习 — 概念讲解、问答练习、知识梳理
  • 头脑风暴 — 点子生成、方案评估、决策支持

用什么方式用 Claude？

  • claude.ai — 网页版，适合对话、分析文件、写文章
  • Claude Code — 终端工具，适合编程开发、项目管理
  • Claude API — 编程接入，适合构建自己的应用
🎯 给 SHARK 的建议： 这个课程侧重"会用"而不是"安装配置"。 如果你还没装 Claude Code，之后需要时再说。 先学会用好网页版和对话技巧，这些在任何场景下都通用。
