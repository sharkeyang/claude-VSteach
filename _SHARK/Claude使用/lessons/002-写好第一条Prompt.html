第 2 课 · 写好第一条 Prompt

Prompt（提示词）是你和 Claude 对话的第一句话——它决定了 Claude 会给你什么质量的回答。

一个好 Prompt 的核心公式

  • 角色 — 你希望 Claude 扮演什么身份
  • 背景 — 当前的任务场景是什么
  • 任务 — 你具体要做什么
  • 格式 — 你希望怎么输出
  • 约束 — 有什么限制条件
❌ 差的 Prompt：「帮我写个方案」
✅ 好的 Prompt：「你是一位有10年经验的产品经理（角色），我正在做一款家庭记账App（背景），请帮我写一份竞品分析框架（任务），用表格列出3个竞品的核心功能对比（格式），每个竞品不超过100字（约束）」

实用技巧

  • 说具体，不要抽象 — "总结这份报告"不如"用3句话总结这份报告的核心观点，每句不超过20字"
  • 一次只做一件事 — 不要在一句话里塞多个要求，Claude 容易抓错重点
  • 给例子 — 想让 Claude 模仿某种风格，给一个例子比描述更有效
  • 让 Claude 思考 — "先分析问题，再给出方案"比直接要答案更准确
  • 不满意就迭代 — 直接说"换个风格"、"再简洁一点"、"更专业一些"
💡 记住： 你给 Claude 的信息质量 = 你获得的答案质量。 和 Claude 说话就像和一位聪明但需要明确指引的同事合作。 疑的地方就问，不对的就纠正——它是助手，不是读心术师。
