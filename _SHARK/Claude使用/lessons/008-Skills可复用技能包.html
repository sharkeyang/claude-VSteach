第 7 课 · Skills — 可复用技能包

Skills 是 Claude Code 的"技能包"——把常用的工作流和规则打包成文件，下次再处理同样任务时直接调用，不用每次都从头说一遍。

Skills 适合什么？

  • 每日生成计划 — 输入日期和素材，自动生成 Word 文档
  • 代码审查 — 每次改动代码后自动检查语法和逻辑
  • 数据校验 — 下载完数据后自动检查完整性
  • 课程生成 — 按固定模板生成教学页面
✅ VSteach 实践：
/chris-daily-plan — 自动生成 Chris 每日计划
/calligraphy — 输入古文名，生成字帖 PDF

Skill 怎么用？

  • 斜杠命令 — 对话中输入 /skill-name（如 /chris-daily-plan）
  • 自然语言 — 说"生成每日计划"也会自动触发 Skill

什么时候该创建 Skill？

  • 有个流程你每周至少做一次
  • 流程步骤固定、变化不大
  • 每次都要重复描述同样的规则
💡 建议： 先把流程用自然语言跑几遍，确认稳定了再封装成 Skill。 频繁变化的工作不适合做 Skill，直接口述更灵活。
