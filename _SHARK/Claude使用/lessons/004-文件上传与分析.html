第 4 课 · 文件上传与分析

Claude 支持多种文件格式的上传，这意味着你可以直接把资料丢给它，让它帮你分析、总结、提取信息。

支持的文件类型

类型格式用途
图片JPG、PNG、GIF、WebPClaude 能"看懂"图片内容，不只是读文字
文档PDF、TXT提取总结、翻译、关键信息提取
OfficeDOCX、XLSX、PPTX通过 Skills 解析处理
代码.py、.js、.html 等代码审查、Debug、重构
数据CSV、JSON数据分析、格式转换

图片分析的妙用

  • 截图 → 代码 — 给 Claude 一张 UI 截图，让它写出对应的 HTML/CSS
  • 手写笔记 → 电子版 — 拍一张手写笔记，让 Claude 整理成结构化文档
  • 图表解读 — 上传 K 线图、流程图，让 Claude 帮你分析含义
  • PDF 合同/报告 — 上传 PDF，让 Claude 提取关键条款、总结要点
💡 实用技巧： 上传图片前，确保图片清晰、文字可辨。 如果图片模糊，Claude 可能看错内容。 对于多页文档，尽量上传 PDF 而不是逐张截图。

文件处理的最佳实践

  • 大文件先拆分 — 超过 100 页的 PDF 可以分批上传，或让 Claude 先读目录
  • 交代任务再上传 — 先告诉 Claude 你要它做什么，再上传文件
  • 多文件场景 — 可以同时上传多个文件，Claude 会综合理解
