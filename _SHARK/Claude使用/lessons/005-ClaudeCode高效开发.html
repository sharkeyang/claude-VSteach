第 6 课 · Claude Code 高效开发

Claude Code 是运行在终端里的 AI 编程助手——你可以直接用它来读写文件、运行命令、管理项目。

📌 本课不讲安装： 如果你还没装 Claude Code，等需要的时候再处理。 这里只讲"它能做什么"和"怎么用它更高效"。

Claude Code 能做什么？

  • 读写文件 — 创建、编辑、查看代码文件
  • 运行命令 — 在终端里执行 Python、Git 等命令
  • 搜索代码 — 在整个项目中搜索关键词、定位问题
  • Git 操作 — 自动提交代码、查看历史、创建分支
  • 项目验证 — 自动运行测试和 Lint 检查

核心使用模式

  • /slash 命令 — 如 /code-review 审查当前代码改动，/clarify 澄清需求
  • 自然语言指令 — 直接说"帮我加个注释"、"重构这个函数"
  • Plan 模式 — 先规划再动手，避免改半天发现方向错了
  • 多文件操作 — 可以同时改多个文件，保持一致性
💡 效率技巧：
  • 在项目根目录创建 CLAUDE.md，Claude Code 每次进入自动读取
  • 复杂任务先说清楚方向，让 Claude Code 出方案你再确认
  • 善用 read、glob、grep 等内置工具快速了解代码
  • 如果 Claude Code 改了多个文件，确认没问题再让它提交

和 claude.ai 的区别

claude.aiClaude Code
对话为主，适合分析、写作操作为主，适合写代码、管理项目

无法运行代码可以直接运行 Python、Git 等命令
