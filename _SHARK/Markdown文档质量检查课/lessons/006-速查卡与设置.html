二、Markdown 语法速查表

语法正确写法常见错误
标题# 标题（有空格）#标题（无空格）
列表- 项目（2空格缩进）缩进不一致
表格| 列 | 列 |列数不匹配
代码块```python...```缺少结束符
引用> 内容忘记 >
粗体**文字****文字*（少一个*）
斜体*文字**文字（未闭合）
链接[text](url)格式错误

三、VS Code 推荐设置

将以下内容添加到 settings.json：

{
  // 自动检测文件编码
  "files.autoGuessEncoding": true,

  // Markdown 预览换行
  "markdown.preview.breaks": true,

  // 显示空格和制表符
  "editor.renderWhitespace": "boundary",

  // 保存时自动格式化
  "editor.formatOnSave": true
}

四、最常用的 3 个快捷键

快捷键用途
Ctrl+Shift+V预览 Markdown 渲染效果
Ctrl+F搜索特殊字符（如零宽空格）
Ctrl+Shift+P → 编码检查/修改文件编码为 UTF-8
