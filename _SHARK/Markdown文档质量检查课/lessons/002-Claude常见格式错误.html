第2课：Claude 常见格式错误 TOP 12

每个错误都附表现、示例、修复方法

1表格列数不一致

表现：表格歪歪扭扭，一列跑到另一列去了。

| 原文 | 解释 | 提醒 | ← 3列
| 好言一句 | 好话让人暖 | 你随口一句 | ← ✅ 3列
| 伤身事莫做 | 伤身体的事 | ← ❌ 只有2列！

修复：确保每行 | 的数量相同。

2表格分隔行格式不对

表现：表格完全不渲染。

| 原文 | 解释 |
--- | --- | --- ← ❌ 分隔行没有 | 开头
✅ |:---|:---| ← 前面有 |，后面有 |

3列表缩进不一致

表现：子列表项跑到外面去了。

- 主列表
  - 子列表（2空格）
 - 这行缩进1空格 ← ❌ 渲染异常

修复：全部用 2 个空格缩进一级。

4代码块未正确闭合

表现：后面所有内容变成灰色代码块。

```python
print("hello")
← ❌ 少了结束的 ```
后面所有内容都变灰色...

检查：搜索 ```，出现次数必须是偶数。

5中英文引号混用

表现：Python 报错 SyntaxError。

scene = "同学跑来跟你说"老师骂你了"他也会说" ← ❌ 引号冲突

修复：将中文引号替换为「」：

scene = '同学跑来跟他说「老师骂你了」他也会说' ← ✅

6<small>标签未闭合

表现：整个页面后面的字体都变小。

<small>这是小字 ← ❌ 没有 </small>

检查：统计 <small> 和 </small> 数量，必须相等。

7文件编码为 GBK 而非 UTF-8

表现：中文显示为乱码，如 ç»Ÿè®¡。

修复：VS Code 右下角点击编码 → 选择「通过编码保存」→ UTF-8。

8零宽空格

表现：表格看起来没问题，但有些行无法对齐。肉眼看不到！

检查：VS Code 中 Ctrl+F，搜索零宽空格字符（U+200B）。

9超长行导致表格溢出

表现：表格需要左右滚动才能看完。

修复：单元格内容控制在 30 字以内，或手动换行。

10标题层级跳级

表现：VS Code 大纲面板出现断层。

# 一级 → ### 三级 ← ❌ 跳过二级

11| 被用在表格之外

表现：普通段落中的 | 被误解析为表格。

修复：在 Markdown 普通段落中使用 | 需要转义 \|。

12URL 中包含空格或中文

表现：链接点击无效。

[链接](https://example.com/my file.pdf) ← ❌ 有空格
✅ [链接](https://example.com/my%20file.pdf)
