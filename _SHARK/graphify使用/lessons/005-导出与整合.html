第 5 课 · 导出与整合

默认产出

每次建图自动生成：

  • graph.json — 原始图谱数据（节点+边+社区）
  • GRAPH_REPORT.md — 审计报告（God Nodes、社区、隐藏关系、知识缺口）
  • graph.html — 交互式可视化

HTML 交互图

默认生成，浏览器直接打开：

graphify export html

超过 5000 节点时自动聚合为社区视图。节点颜色按社区区分，可拖拽浏览。

SVG 导出

/graphify D:\project --svg

导出 graph.svg，可嵌入 Notion、GitHub README、文档。

GraphML 导出

/graphify D:\project --graphml

导出 graph.graphml，可导入 Gephi 或 yEd 做更专业的图分析。

Neo4j 导出/推送

/graphify D:\project --neo4j              # 导出 Cypher 语句
/graphify D:\project --neo4j-push bolt://localhost:7687  # 直接推入

导出 graphify-out/cypher.txt，可直接导入 Neo4j 图数据库。

Obsidian 知识库

/graphify D:\project --obsidian
/graphify D:\project --obsidian --obsidian-dir ~/vaults/my-project

每个节点生成一个 Obsidian 笔记文件，社区用文件夹区分。默认输出到 graphify-out/obsidian/。

FalkorDB 导出

/graphify D:\project --falkordb
/graphify D:\project --falkordb-push falkordb://localhost:6379

Wiki 导出

/graphify D:\project --wiki

生成可浏览的 Wiki 页面，每个社区一篇文章，方便分享给非技术用户。

MCP 服务器

/graphify D:\project --mcp

启动 MCP stdio 服务器，让其他 AI 工具也能访问这个图谱。

导出格式速查

格式产出文件用途
默认graph.html浏览器交互浏览
--svggraph.svg嵌入文档/Notion/GitHub
--graphmlgraph.graphmlGephi/yEd 专业分析
--neo4jcypher.txt导入 Neo4j 图数据库
--obsidianobsidian/Obsidian 知识库
--wikiwiki/可浏览的静态 Wiki
💡 推荐： 日常用默认 HTML 浏览 + GRAPH_REPORT.md 查看报告。需要深度分析时用 GraphML 导入 Gephi。
