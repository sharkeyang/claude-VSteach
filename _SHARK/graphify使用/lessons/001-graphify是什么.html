第 1 课 · graphify 是什么

知识图谱是什么？

知识图谱把文件中的实体（函数、类、概念、文档）变成节点，把它们之间的关系（调用、引用、依赖、概念相关）变成边。结果是一个可导航、可查询的网络，而不是一堆孤立文件。

graphify 能做什么

  • 全格式支持 — 代码(.py/.ts/.go/...)、文档(.md/.txt)、PDF、图片、视频/音频
  • 自动提取 — 代码走 AST 解析（免费、无 LLM），文档走 LLM 语义提取
  • 社区检测 — 自动把相关节点聚成社区，发现跨文件的隐藏联系
  • 自然语言查询 — 图建好后，用日常语言提问，图谱遍历找答案
  • 增量更新 — 改文件后只用 --update，不用全量重跑
  • 多种导出 — HTML 交互图、SVG、GraphML、Neo4j、Obsidian

安装

pip install graphifyy
graphify install

注册后，Claude Code 中输入 /graphify 即可使用。

运行原理

  1. 检测 — 扫描目录，分类文件（代码/文档/PDF/图片/视频）
  2. AST 提取 — 代码文件用语法树自动解析，零成本
  3. 语义提取 — 文档/PDF/图片由 LLM 提取概念和关系
  4. 建图+聚类 — 合并结果，构建图网络，自动社区检测
  5. 报告+导出 — 生成 GRAPH_REPORT.md 和 HTML 可视化
💡 不需要 API Key： 纯代码项目完全免费（AST 解析）。文档项目用 Claude 自身的子 agent 提取，也不需要额外 API Key。如需加速可设置 GEMINI_API_KEY。

什么时候值得用

  • 项目文件超过 20 个，手动 grep 已不够用
  • 需要跨文件理解架构和概念关系
  • 整理学习资料，想看清知识之间的联系
  • 新接手代码库，想快速上手
