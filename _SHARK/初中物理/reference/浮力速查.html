🌊 浮力 · 速查

第十章 阿基米德的浴缸 · 一页纸搞定浮力

📜 起源（阿基米德，公元前3世纪）： 国王让查王冠是否纯金→阿基米德泡澡看到水溢出→顿悟"浮力=排开水的重量"→光着身子大喊"Eureka!"

💡 浮力的本质

产生原因： 上下表面液体压力差（下表面更深→压强更大→压力更大→向上推）
测量方法： 称重法 F浮 = G − F测

⚖️ 阿基米德原理

F浮 = G排 = ρ液 g V排

  • 浮力只与 ρ液（液体密度）和 V排（排开体积）有关
  • 与物体形状、材质、浸没深度（完全浸没后）无关
  • V排 ≠ V物（漂浮时 V排 < V物）

⬆️⬇️ 浮沉条件

条件运动最终状态
F浮 > G上浮漂浮（F浮 = G，ρ物<ρ液）
F浮 = G悬浮悬浮（ρ物 = ρ液）
F浮 < G下沉沉底（ρ物 > ρ液）

🚢 应用：一句话

  • 轮船： 空心增大 V排 → 增大 F浮
  • 潜水艇： 注水/排水改变 G（F浮不变）
  • 热气球： 加热空气 → 平均 ρ 变小 → ρ平均<ρ空气
  • 盐水选种： 饱满种子 ρ 大→沉；干瘪种子 ρ 小→浮
🥷 秒杀技巧：轮船从江入海
浮力 不变（始终漂浮，F浮=G）→ ρ海 > ρ江 → V排 减小 → 船上浮一些
考试每年都有人选"浮力增大"——记住：漂浮永远 F浮=G，G不变→F浮不变
🧠 第一性原理追问：为什么钢铁能浮在水上？
单一逻辑链：
做成空心 → 同样质量下占据更大空间 → 平均密度 ρ平均 = m/V总 变小 → ρ平均 < ρ水 → 漂浮。
关键是"平均密度"——不是铁本身的密度，而是整艘船的密度（含内部空气）。
