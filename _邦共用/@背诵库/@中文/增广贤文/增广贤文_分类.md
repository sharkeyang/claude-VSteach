# 《增广贤文》全文归类分析

> 针对：大宝（Chris）—— 口无遮拦、不懂人情世故、不会人际关系、做事不认真
> 归类原则：按重要性排序（对 Chris 最需要学的内容排最前）

---

## 🏆 第一优先：说话之道（针对「口无遮拦」）

> **大宝问题：** 经常口无遮拦，想什么说什么，不懂分寸。以下每句背过之后，遇到类似场景要能想起来用。

| **原文** | <small>一句话解释</small> | <small>🎯 大宝提醒</small> |
|:---|:---|:---|
| **好言一句三冬暖，恶语伤人六月寒** | <small>好话让人冬天都暖，坏话让人夏天都寒</small> | <small>你随口一句伤人的话，别人可能记很久。说话前停三秒。</small> |
| **伤身事莫做，伤心话莫说** | <small>伤身体的事不做，伤人心的话不说</small> | <small>你有时候说话像刀子，说完自己忘了，别人心里留了疤。</small> |
| **过头饭好吃，过头话难听** | <small>饭做多了可以吃，话说过了就难听</small> | <small>别把话说太满、说太绝，留点余地。</small> |
| **好话不在多说，有理不在高声** | <small>好话不在多，有理不用大声喊</small> | <small>你有时候越说越激动、声音越大，反而显得没理。</small> |
| **蚊虫遭扇打，只因嘴伤人** | <small>蚊子被打是因为它叮人，嘴伤人也会招祸</small> | <small>想想你上次因为说话太冲吃了亏的事。</small> |
| **祸从口出，病从口入** | <small>灾祸从嘴里出来，病从嘴里进去</small> | <small>嘴是福祸的门。不该说的话，咽回去。</small> |
| **口开神气散，舌出是非生** | <small>一开口精气神就散了，舌头一动是非就来了</small> | <small>少说废话，多听多想。话多耗神还惹事。</small> |
| **一言既出，驷马难追** | <small>一句话说出口，四匹马都追不回</small> | <small>话说了就收不回了，想说之前先问自己：这话该说吗？</small> |
| **说长说短，宁说人长莫说短** | <small>要说就说人好处，别说人短处</small> | <small>你有时候喜欢评论别人。说人好话积德，说人坏话树敌。</small> |
| **平生只会说人短，何不回头把己量** | <small>整天说别人短处，怎么不看看自己？</small> | <small>说别人之前，先看看自己有没有同样的问题。</small> |
| **当面论人惹恨最大，是与不是随他说吧** | <small>当面说人最招恨，对错让人家说去</small> | <small>别当面指出别人的毛病，尤其是在人多的场合。</small> |
| **好言难得，恶语易施** | <small>好话不容易说出口，坏话却很容易讲</small> | <small>说好话要练习，说坏话不用学。你选哪个？</small> |
| **伤人一语，利如刀割** | <small>伤人的一句话，像刀割一样疼</small> | <small>你说话直，有时候自己觉得"我只是说实话"，但别人觉得被捅了一刀。</small> |
| **是非只为多开口，烦恼皆因强出头** | <small>是非是因为话多，烦恼是因为爱出头</small> | <small>少说话、少出头，烦恼自然少。</small> |
| **会说说都是，不会说无理** | <small>会说话的人怎么说都有理，不会说的人有理也变没理</small> | <small>不是不让你说话，是要学着说。同样的话，换个说法效果完全不同。</small> |
| **知事少时烦恼少，识人多处是非多** | <small>知道的事少烦恼就少，认识的人多是非就多</small> | <small>不该打听的别打听，不该插嘴的别插嘴。</small> |
| **言轻莫劝人** | <small>说话没分量就别去劝人</small> | <small>你年纪小、资历浅，有些场合轮不到你说话。</small> |
| **能言不是真君子，善处方为大丈夫** | <small>能说会道不一定是君子，会办事才是真本事</small> | <small>少耍嘴皮子，多办实事。</small> |
| **贵人语少，贫子话多** | <small>有地位的人话少，贫贱的人话多</small> | <small>话多显得轻浮。你看那些厉害的人，都是话少但句句在点。</small> |
| **一字入公门，九牛拔不出** | <small>一旦告到官府，九头牛都拉不回来</small> | <small>说话要谨慎，有些话说出去就收不回来了。</small> |
| **事非亲见，切莫乱谈** | <small>不是自己亲眼看到的，不要乱说</small> | <small>你没亲眼看到的、不确定的事，别乱传。</small> |
| **打人莫打脸，骂人莫骂短** | <small>打人不打脸，骂人不揭短</small> | <small>吵架也别揭人伤疤，那是结死仇。</small> |
| **话不投机半句多** | <small>说不到一块去，半句都多余</small> | <small>发现跟人聊不来，就闭嘴，别硬聊。</small> |
| **宁在人前全不会，莫在人前会不全** | <small>宁可让人知道你完全不会，也别让人知道你会一点点</small> | <small>不懂就不懂，别装懂。一开口就露馅更丢人。</small> |
| **会打三班鼓，也要几个人** | <small>再厉害的人也需要帮手</small> | <small>别觉得自己什么都行，说话做事别太狂。</small> |
| **酒中不语真君子，财上分明大丈夫** | <small>喝酒不乱说话是真君子，钱财上清楚才是大丈夫</small> | <small>喝了酒更要管住嘴，别借酒劲乱说话。</small> |
| **知音说与知音听，不是知音莫与谈** | <small>懂的人说给懂的人听，不是知音就别谈</small> | <small>不是所有人都值得你掏心窝子，看人说话。</small> |
| **逢人且说三分话，未可全抛一片心** | <small>跟人说话留三分，别把心全掏出来</small> | <small>你不是不懂人情世故吗？那就先从"少说"开始。</small> |
| **莫道坐中安乐少，须知世上苦情多** | <small>别总觉得自己不快乐，世上苦的人多着呢</small> | <small>少抱怨，多感恩。</small> |

## 🥈 第二优先：处世智慧（针对「不懂人情世故」）

> **大宝问题：** 不懂人情世故，不会看场合、看脸色。以下教他基本的处世规则。

| **原文** | <small>一句话解释</small> | <small>🎯 大宝提醒</small> |
|:---|:---|:---|
| **出门看天色，进门看脸色** | <small>出门看天气，进门看别人的表情</small> | <small>到别人家先看主人脸色，忙就别打扰，高兴再说话。</small> |
| **人在家中坐，祸从天上落** | <small>人在家坐着，灾祸也可能从天而降</small> | <small>要有风险意识，别觉得坏事不会发生在自己身上。</small> |
| **晴天不肯去，直待雨淋头** | <small>晴天不去，等雨淋头才去</small> | <small>该做的事早点做，别拖到最后一刻。</small> |
| **成人不自在，自在不成人** | <small>想成才就别想舒服，想舒服就别想成才</small> | <small>你想要轻松，就别怪自己没进步。</small> |
| **来说是非者，便是是非人** | <small>跑来跟你说别人是非的人，他自己就是是非人</small> | <small>如果有人在你面前说别人坏话，他也会在别人面前说你坏话。</small> |
| **墙有缝，壁有耳** | <small>墙有缝会透风，墙外有耳朵在听</small> | <small>你觉得是悄悄话，可能已经被别人听去了。别在背后说人。</small> |
| **若要人不知，除非己莫为** | <small>不想让人知道，就别做</small> | <small>做了就别怕人知道。想做坏事时想想这句。</small> |
| **为人不做亏心事，半夜敲门心不惊** | <small>不做亏心事，就不怕鬼敲门</small> | <small>做人坦荡，心里就踏实。</small> |
| **万事劝人休瞒昧，举头三尺有神明** | <small>别瞒着良心做事，头顶三尺有神明看着</small> | <small>别以为没人看见就可以做坏事。</small> |
| **害人之心不可有，防人之心不可无** | <small>不害人，但要防人</small> | <small>你不懂人情世故，更要学会保护自己。</small> |
| **知人知面不知心** | <small>认识一个人的脸，不认识他的心</small> | <small>别太快相信一个人，要看他的行动。</small> |
| **路遥知马力，日久见人心** | <small>路远才知道马力，时间久了才看清人心</small> | <small>朋友不是一天交出来的，时间会告诉你谁是真的对你好。</small> |
| **画虎画皮难画骨，知人知面不知心** | <small>画老虎画皮容易画骨难，认识人认识脸难认识心</small> | <small>人不可貌相，别凭第一印象判断人。</small> |
| **近水知鱼性，近山识鸟音** | <small>靠近水才知道鱼的习性，靠近山才认识鸟的声音</small> | <small>多接触人、多观察，你才能学会人情世故。</small> |
| **相识满天下，知心能几人** | <small>认识的人满天下，知心的有几个？</small> | <small>朋友贵在质量，不在数量。</small> |
| **路逢险处须回避，事到临头不自由** | <small>遇到危险要避开，事到临头就由不得你了</small> | <small>有风险的事，提前避开，别等出事才后悔。</small> |
| **忍一句，息一怒，饶一着，退一步** | <small>忍一句、消一次气、让一步、退一步</small> | <small>跟人起冲突时，先退一步，不吃亏。</small> |
| **得忍且忍，得耐且耐，不忍不耐，小事成灾** | <small>能忍就忍，能让就让，不忍会让小事变大祸</small> | <small>你脾气上来了就容易冲动。忍一下，过五分钟再看。</small> |

| **近来学得乌龟法，得缩头时且缩头** | <small>学乌龟，该缩头就缩头</small> | <small>不是什么时候都要硬碰硬，有时候退就是进。</small> |
| **受恩深处宜先退，得意浓时便可休** | <small>受恩太多要懂得退，得意太浓要懂得收</small> | <small>别贪，别得意忘形。见好就收。</small> |

| **莫待是非来入耳，从前恩爱反为仇** | <small>别等是非传到耳朵里，从前的恩爱变成了仇</small> | <small>朋友之间别听信传言，有问题直接问。</small> |
| **留得五湖明月在，不愁无处下金钩** | <small>只要五湖明月还在，不怕没地方钓鱼</small> | <small>只要身体健康、本事在，不怕没机会。</small> |
| **休别有鱼处，莫恋浅滩头** | <small>别离开有鱼的地方，别留恋浅滩</small> | <small>珍惜已有的好条件，别这山望着那山高。</small> |
| **留得青山在，不怕没柴烧** | <small>只要山还在，不怕没柴烧</small> | <small>身体和命最重要，其他都是次要的。</small> |
| **知足常足，终身不辱** | <small>知足就永远满足，终身不受耻辱</small> | <small>别总是跟人比，比来比去自己难受。</small> |
| **知止常止，终身不耻** | <small>知道适可而止，终身不受耻辱</small> | <small>该停就停，别贪心。</small> |
| **用心计较般般错，退后思量事事宽** | <small>斤斤计较样样错，退一步想想路就宽了</small> | <small>别什么事都计较对错，退一步海阔天空。</small> |
| **当时若不登高望，谁信东流海洋深** | <small>当时不登高远望，谁会相信东流的海那么深？</small> | <small>站得高才能看得远。别只盯着眼前。</small> |
| **自恨枝无叶，莫怨太阳偏** | <small>怪自己树枝没叶子，别怪太阳偏了</small> | <small>有问题先找自己的原因，别怪别人。</small> |
| **自家心里急，他人未知忙** | <small>自己心里急，别人不知道你的忙</small> | <small>别指望别人理解你的难处，自己的事自己扛。</small> |
| **人各有心，心各有见** | <small>每个人都有自己的想法和看法</small> | <small>别要求别人跟你想的一样。</small> |
| **凡人不可貌相，海水不可斗量** | <small>不能以貌取人，海水不能用斗量</small> | <small>别因为别人的外表、穿着就判断人家。</small> |
| **路不铲不平，事不为不成** | <small>路不铲不平，事不做不成</small> | <small>光想不做，什么都不会改变。</small> |
| **河狭水激，人急计生** | <small>河道窄水流急，人急了会想出办法</small> | <small>急的时候别慌，办法会有的。</small> |

## 🥉 第三优先：人际关系（针对「不会人际关系」）

> **大宝问题：** 不会跟人相处，不懂得维系关系。以下教他交友和待人的基本道理。

| **原文** | <small>一句话解释</small> | <small>🎯 大宝提醒</small> |
|:---|:---|:---|
| **知己知彼，将心比心** | <small>了解自己也了解别人，换位思考</small> | <small>跟人相处时，想想如果你是对方，你希望别人怎么对你。</small> |
| **责人之心责己，恕己之心恕人** | <small>用责备别人的心来责备自己，用原谅自己的心来原谅别人</small> | <small>对别人宽容点，对自己严格点。你正好反过来。</small> |
| **宁可人负我，切莫我负人** | <small>宁可别人辜负我，我不能辜负别人</small> | <small>别做对不起人的事。</small> |
| **再三须慎意，第一莫欺心** | <small>做事要再三谨慎，第一不要欺骗自己的良心</small> | <small>对得起自己的良心，别人怎么说是别人的事。</small> |
| **远水难救近火，远亲不如近邻** | <small>远处的救不了近火，远亲不如近邻</small> | <small>跟身边的人搞好关系，关键时刻邻居比远亲管用。</small> |
| **和得邻里好，犹如拾片宝** | <small>跟邻居处得好，就像捡到宝</small> | <small>见到邻居打个招呼，笑脸相迎不吃亏。</small> |
| **只有和气去迎人，哪有相打得太平** | <small>只有和气待人才能太平，打架解决不了问题</small> | <small>有矛盾好好说，别动手、别吵架。</small> |
| **父子和而家不退，兄弟和而家不分** | <small>父子和睦家不衰，兄弟和睦家不分</small> | <small>家人之间更要和睦，对外人客气对家人发脾气最蠢。</small> |
| **夫妻相和好，琴瑟与笙簧** | <small>夫妻和睦就像琴瑟和鸣</small> | <small>以后你结婚了要记住：家和万事兴。</small> |
| **求人须求大丈夫，济人须济急时无** | <small>求人要找真正有担当的人，帮人要帮在急处</small> | <small>帮人要雪中送炭，别锦上添花。</small> |
| **渴时一滴如甘露，醉后添杯不如无** | <small>渴的时候一滴水像甘露，喝醉了再添酒不如不添</small> | <small>要在别人最需要的时候帮忙。</small> |
| **与人方便自己方便** | <small>给别人方便，就是给自己方便</small> | <small>帮别人就是帮自己。</small> |
| **当面留一线，过后好相见** | <small>当面留点余地，以后好再见面</small> | <small>不管跟谁，别把关系做绝了。</small> |
| **结交须胜己，似我不如无** | <small>交朋友要找比自己强的，不如自己的不如不交</small> | <small>跟优秀的人在一起，你也会变优秀。</small> |
| **三人行必有我师焉** | <small>三个人一起走，一定有可以当我老师的人</small> | <small>每个人都有值得你学的地方。</small> |
| **择其善者而从之，其不善者而改之** | <small>选好的学，不好的反省自己有没有</small> | <small>看到别人好的地方学，看到不好的提醒自己别那样。</small> |
| **君子之交淡如水** | <small>君子之间的交情清淡如水</small> | <small>真正的朋友不是天天腻在一起，而是有事时能靠得住。</small> |
| **有酒有肉多兄弟，急难何曾见一人** | <small>有酒肉的时候兄弟多，有急难的时候一个人都没有</small> | <small>酒肉朋友靠不住。真朋友是平时不常联系，但你有事他一定到。</small> |
| **人情似纸张张薄，世事如棋局局新** | <small>人情薄如纸，世事新如棋</small> | <small>别太指望别人，靠自己最实在。</small> |
| **人情似水分高下，世事如云任卷舒** | <small>人情像水有高有低，世事像云有卷有舒</small> | <small>人情冷暖是正常的，别太在意。</small> |
| **贫居闹市无人问，富在深山有远亲** | <small>穷了住在闹市也没人理，富了住在深山也有人来认亲</small> | <small>现实就是这样。你要努力让自己变得有价值。</small> |
