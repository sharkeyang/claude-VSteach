🔄 PDCA 循环

第2课 · 计划→执行→检查→改进

来源： Shewhart & Deming，质量管理核心方法
核心思想： 任何工作都可以通过循环不断优化。
不是一次做对，而是每次循环都发现一个改进点，日积月累进步惊人。

四步详解

P — Plan（计划）

  • 目标要具体（SMART）
  • 方案要可执行
  • 先问：我想达到什么效果？

D — Do（执行）

  • 按计划去做
  • 记录过程和数据
  • 不要边做边改（那是下一轮的事）

C — Check（检查）

  • 目标达成多少？
  • 哪里做得好？哪里不好？
  • 对比数据，不要凭感觉

A — Act（改进）

  • 找到根本原因
  • 修改计划，进入下一循环

🏀 场景案例：打篮球投篮

第1轮

步骤内容
P每天练投篮20次，目标是投进10个
D投了20次，只进了4个
C命中率20%，投篮姿势不稳定
A下一轮专门练"手肘对正"

第2轮： 先练10次"手肘对正"姿势，再投20个 → 进了7个（35%） → 手腕发力不对 → 加练"手腕下压"

第3轮起： 每次改一个点……一个月后命中率从20%提高到60%。
PDCA的力量不是一次改很多，而是每次改一个点，一圈一圈积累。

📖 场景案例：背诵古文

第1轮： 计划用3天背《木兰辞》→ 每天读5遍 → 只能背前半段 → 问题是一次性读全文 → 改进：分段背

第2轮： 分成4段，每天背1段+复习前1段 → 能完整背出，但3处卡顿 → 改进：专攻卡顿句

第3轮： 3句卡顿每句背10遍 → 完整背出零卡顿 → 结论：分段+专攻难点是最有效的背诵方法

📜 场景案例：学习历史

第1轮： 计划一周读完"三国"→ 读了记不住，时间线混乱 → 改进：画时间线+因果关系

第2轮： 画时间线（黄巾→群雄→官渡→赤壁→三国鼎立）→ 时间线清了，人物关系混乱 → 改进：加人物关系图

第3轮： 画人物关系图（曹操-刘备-孙权关系，谋士和将领）→ 时间线和人物都清晰了 → 结论：学历史=时间线+人物关系图

🧮 场景案例：深度思考（学而思数学）

第1轮： 做5道分数应用题，全对为目标 → 错了2道 → 都是"单位1"找错 → 改进：解题前先圈"单位1"

第2轮： 做5道新题，先圈"单位1" → 全对！但速度慢 → 改进：练习快速定位单位1

💬 场景案例：回答问题 & 汇报

第1轮： 做"我最喜欢的动物"汇报，计划讲3分钟 → 讲了1分半，内容少语速快 → 改进：总分总结构+先练2遍

第2轮： "我的暑假"汇报，总分总结构+练2遍 → 讲了3分钟结构清晰，但手抖 → 改进：练习站定+深呼吸

第3轮： 注重站姿和眼神交流，让妈妈当观众练习 → 获老师表扬！→ 结论：好的汇报=内容丰富+结构清晰+练习充分

🤝 场景案例：人际交际

第1轮： 本周和XX同学做朋友 → 主动说话但只说了"你好" → 没话题尴尬 → 改进：准备3个话题

第2轮： 聊他喜欢的动画片，聊了20分钟 → 聊得开心但总是我问 → 改进：双向分享

第3轮： 分享+提问结合，聊了周末打篮球 → 他也会打，约了一起打球 → 结论：交朋友=主动+准备话题+双向分享

日常应用汇总

场景P计划D执行C检查A改进
🏀 投篮每天投20中10投了20中4姿势不对练手肘对正
📖 背古文3天背完背了一半后半段记不住分段背
📜 学历史读完三国读了记不住时间线混乱画时间线+人物图
🧮 数学题5题全对错了2题单位1找错先圈单位1
💬 汇报讲3分钟讲了1分半内容太少总分总+练习
🤝 交朋友和XX聊天只说了你好没话题准备话题

核心要点

  • P 不等于完美——计划可以粗，但要可执行
  • D 要忠实执行——不要边做边改方案
  • C 要诚实——好就是好，不好就是不好
  • A 最关键——不改进，前面的都白做
  • 整个循环可以很快——5分钟的PDCA也有用
把 PDCA 变成习惯：每做完一件事，花2分钟问"下次怎么做得更好？"
