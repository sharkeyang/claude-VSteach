# 🔄 PDCA 循环

> 第2课 · 计划→执行→检查→改进

---

## 来源

**Shewhart & Deming**，质量管理核心方法

## 核心思想

任何工作都可以通过"计划→执行→检查→改进"的循环不断优化。

> 不是一次做对，而是每次循环都发现一个改进点。日积月累，进步惊人。

---

## 四步详解

### P — Plan（计划）
- 目标要具体（SMART）
- 方案要可执行
- 先问：我想达到什么效果？

### D — Do（执行）
- 按计划去做
- 记录过程和数据
- 不要边做边改（那是下一轮的事）

### C — Check（检查）
- 目标达成多少？
- 哪里做得好？哪里不好？
- 对比数据，不要凭感觉

### A — Act（改进）
- 找到根本原因
- 修改计划，进入下一循环

## 🏀 场景案例：打篮球投篮

### PDCA 第1轮：先试试

| 步骤 | 内容 |
|:----|:------|
| **P** | 计划：每天练投篮20次，目标是投进10个 |
| **D** | 执行：第一天投了20次，只进了4个 |
| **C** | 检查：命中率20%，远低于目标；发现投篮姿势不稳定 |
| **A** | 改进：下一轮专门练"手肘对正"这个环节 |

### PDCA 第2轮：针对性练习

| 步骤 | 内容 |
|:----|:------|
| **P** | 计划：先练10次"手肘对正"的姿势，再投20个 |
| **D** | 执行：姿势练完再投，进了7个 |
| **C** | 检查：命中率35%，有进步但还不够；发现手腕发力不对 |
| **A** | 改进：下一轮加入"手腕下压"练习 |

### 第3轮、第4轮……
一个月后，命中率从20%提高到60%。

> PDCA 的力量不是一次改很多，而是**每次改一个点，一圈一圈积累**。

## 📖 场景案例：背诵古文

### 第1轮
| 步骤 | 内容 |
|:----|:------|
| **P** | 计划：用3天背完《木兰辞》 |
| **D** | 执行：每天读5遍，3天后试着背 |
| **C** | 检查：能背出前半段，后半段记不住 |
| **A** | 改进：发现问题是"一次性读全文"，后半段读的次数不够 → 分段背 |

### 第2轮
| **P** | 计划：分成4段，每天背1段 |
| **D** | 执行：每天背1段+复习前1段 |
| **C** | 检查：5天后能完整背出，但有3处卡顿 |
| **A** | 改进：卡顿处单独标出来，专攻这3句 |

### 第3轮
| **P** | 计划：重点背3处卡顿句，每句背10遍 |
| **D** | 执行：专攻3天后 |
| **C** | 检查：完整背出，零卡顿 |
| **A** | 结论：分段+专攻难点是最有效的背诵方法，以后都用这个流程 |

## 📜 场景案例：学习历史

### 第1轮：学"三国时期"
| 步骤 | 内容 |
|:----|:------|
| **P** | 计划：一周内读完"三国"章节，记住重要事件 |
| **D** | 执行：从头到尾读了一遍 |
| **C** | 检查：读了但记不住，时间线混乱（先有官渡之战还是赤壁之战？） |
| **A** | 改进：下一轮用"时间线+因果关系"来学，不要光读 |

### 第2轮
| **P** | 计划：画时间线，每件事写"原因→事件→结果" |
| **D** | 执行：画了三国时间线：黄巾起义→群雄割据→官渡之战→赤壁之战→三国鼎立 |
| **C** | 检查：时间线清楚了，但人物的关系还是混乱 |
| **A** | 改进：下一轮加入"人物关系图" |

### 第3轮
| **P** | 计划：画人物关系图（连线+关系说明） |
| **D** | 执行：画了曹操-刘备-孙权的关系，以及各自的重要谋士和将领 |
| **C** | 检查：现在时间线和人物都清晰了，能讲出完整故事 |
| **A** | 结论：学历史 = 时间线 + 人物关系图，以后的历史章节都用这个框架 |

> 迭代3次就从"读完了记不住"变成"能讲出完整三国故事"。

## 🧮 场景案例：深度思考（学而思数学）

### 第1轮：做分数应用题
| 步骤 | 内容 |
|:----|:------|
| **P** | 计划：做完5道分数应用题，目标是全对 |
| **D** | 执行：做了5道，错了2道 |
| **C** | 检查：错的两道都是"单位1"找错了 |
| **A** | 改进：下一轮解题前先圈出"单位1" |

### 第2轮
| **P** | 计划：做5道新题，解题前先圈"单位1" |
| **D** | 执行：做了5道，全对 |
| **C** | 检查：方法有效！但做题速度有点慢 |
| **A** | 改进：下一轮练习"快速定位单位1"的技巧 |

## 💬 场景案例：回答问题 & 汇报

### PDCA 提升汇报能力
| 步骤 | 内容 |
|:----|:------|
| **P** | 计划：做"我最喜欢的动物"主题汇报，要讲3分钟 |
| **D** | 执行：讲了1分半就没话说了，一直在说"猫很可爱" |
| **C** | 检查：内容太少，没有结构；紧张导致语速太快 |
| **A** | 改进：下次用"总-分-总"结构写提纲，在家先练讲2遍 |

### 第2轮
| **P** | 计划："我的暑假"汇报，用总分总结构，先练2遍 |
| **D** | 执行：写了提纲（总：暑假很开心→分：海边/读书/打球→总：收获），练了2遍再上台 |
| **C** | 检查：讲了3分钟，结构清晰，但手一直在抖 |
| **A** | 改进：下次练习时加上"站定+深呼吸"来缓解紧张 |

### 第3轮
| **P** | 计划：新汇报，注重站姿和眼神交流 |
| **D** | 执行：练习时让妈妈当观众，练习"看着人说" |
| **C** | 检查：汇报获得老师表扬！手还是有点抖但不明显了 |
| **A** | 结论：好的汇报=内容丰富+结构清晰+练习充分 |

## 🤝 场景案例：人际交际

### PDCA 改善和同学的相处
| 步骤 | 内容 |
|:----|:------|
| **P** | 计划：本周想和XX同学成为好朋友 |
| **D** | 执行：下课主动找他说话，但只说了"你好"就没话说了 |
| **C** | 检查：主动了但不知道怎么聊下去，尴尬 |
| **A** | 改进：准备3个话题（他喜欢的动画片、周末做了什么、一起玩什么） |

### 第2轮
| **P** | 计划：用准备的3个话题，聊一次15分钟以上 |
| **D** | 执行：聊了他喜欢的动画片，聊了20分钟 |
| **C** | 检查：聊得很开心，但发现总是我在问问题，他没问我 |
| **A** | 改进：下次可以分享自己的事，不只是问对方 |

### 第3轮
| **P** | 计划：分享+提问结合，让对方也了解我 |
| **D** | 执行：聊了周末打篮球的事，他说他也会打，约了一起打球 |
| **C** | 检查：互相了解了，还约了下次一起玩 |
| **A** | 结论：交朋友 = 主动 + 准备话题 + 双向分享 |

## 日常应用汇总

| 场景 | P 计划 | D 执行 | C 检查 | A 改进 |
|:----|:------|:-------|:-------|:-------|
| 🏀 投篮 | 每天投20中10 | 投了20中4 | 姿势不对 | 练手肘对正 |
| 📖 背古文 | 3天背完《木兰辞》 | 背了一半 | 后半段记不住 | 分段背 |
| 📜 学历史 | 读完三国章节 | 读了记不住 | 时间线混乱 | 画时间线+人物图 |
| 🧮 数学题 | 5题全对 | 错了2题 | 单位1找错 | 先圈单位1 |
| 💬 汇报 | 讲3分钟 | 讲了1分半 | 内容太少 | 总分总结构+练习 |
| 🤝 交朋友 | 和XX聊天 | 只说了你好 | 没话题 | 准备话题 |
| 📝 写作文 | 写300字 | 写了200字 | 内容空洞 | 先列提纲再写 |

---

## 核心要点

- **P** 不等于完美——计划可以粗，但要可执行
- **D** 要忠实执行——不要边做边改方案
- **C** 要诚实——好就是好，不好就是不好
- **A** 最关键——不改进，前面的都白做
- 整个循环可以很快——**5分钟的PDCA也有用**

> 把 PDCA 变成习惯：每做完一件事，花2分钟问"下次怎么做得更好？"
