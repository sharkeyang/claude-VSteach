# ⏰ 番茄工作法

> 第7课 · 25+5 的时间分块法

---

## 提出者

**Francesco Cirillo**，1980年代

## 核心思想

与其强迫自己长时间集中，不如把时间分成小块，每块高度集中后彻底放松。

> 番茄钟用"短时间冲刺+短暂休息"的模式，让大脑保持高效。  
> 25分钟不长，大脑不会抗拒；5分钟的休息，足够恢复精力。

---

## 基本规则

```
① 设25分钟计时器 → ② 专注直到铃响 → ③ 休息5分钟
→ ④ 每4个番茄钟休息15分钟
```

### 重要原则
- 一个番茄钟不可分割
- 中途打断则作废重来
- 休息时不要刷手机
- 番茄钟是时间单位，不是任务

---

## 🏀 场景案例：打篮球

### 用番茄钟练球
| 番茄钟 | 内容 |
|:------|:------|
| 🍅 第1个 | 投篮练习25分钟（投50个，记录命中数） |
| 🍅 第2个 | 运球练习25分钟（左右手各10分钟+变向5分钟） |
| 🍅 第3个 | 防守步伐25分钟（滑步+交叉步） |
| 🍅 第4个 | 实战模拟25分钟（跟爸爸打半场） |
| ⏸ **大休息** | 15分钟，喝水+拉伸 |

**效果：** 以前练1个半小时太累不想练，现在分成4个25分钟+大休息，更有动力。

### 用番茄钟分析比赛
比赛时其实也是"番茄钟模式"——每个进攻回合就是一个"小番茄"：
- 进攻25秒（专注执行战术）→ 休息（对方进攻时放松）→ 下一个进攻
- 和番茄钟的理念一样：**冲刺+恢复，交替进行**

## 📖 场景案例：背诵

### 用番茄钟背古文
| 番茄钟 | 内容 |
|:------|:------|
| 🍅 第1个 | 先理解全文意思（读白话翻译+分段） |
| 🍅 第2个 | 背第1-2段（每段读3遍+背1遍） |
| 🍅 第3个 | 背第3-4段（每段读3遍+背1遍） |
| 🍅 第4个 | 完整背一遍+标记卡顿处 |
| ⏸ **大休息** | 15分钟，吃水果 |

**效果：** 以前背30分钟就坐不住，现在25分钟一歇，节奏感很强。

### 背英语单词
- **1个番茄钟** = 背15个单词（每个读3遍+抄1遍+造句1遍）
- **休息5分钟** = 离开书桌，去喝水
- **再来1个番茄钟** = 复习刚才的15个+背新15个

**效果：** 2个番茄钟背30个，比死磕1小时效率高。

## 📜 场景案例：学习历史

### 用番茄钟学历史
| 番茄钟 | 内容 |
|:------|:------|
| 🍅 第1个 | 读课本三国章节，画时间线 |
| 🍅 第2个 | 看配套视频，记录关键事件 |
| 🍅 第3个 | 用自己的话讲一遍（录制语音） |
| 🍅 第4个 | 做5道练习题+复习错题 |

**效果：** 4个番茄钟就能完成"读→看→讲→练"的完整学习循环。

## 🧮 场景案例：深度思考

### 用番茄钟做学而思数学
| 番茄钟 | 内容 |
|:------|:------|
| 🍅 第1个 | 做3道应用题（不查答案，自己思考） |
| 休息 | 5分钟，站起来活动 |
| 🍅 第2个 | 检查答案+改正+分析错因 |
| 休息 | 5分钟 |
| 🍅 第3个 | 再做2道同类题，巩固 |

**效果：** 把"做作业"变成"番茄钟闯关"，完成一个番茄钟有成就感。

### 深度思考练习
遇到难题时，**先给自己1个番茄钟的时间思考**：
- 前5分钟：读题+画图
- 中间10分钟：尝试不同方法
- 最后10分钟：如果还是不会，标记下来问人

**效果：** 以前遇到难题直接放弃或马上问人，现在至少自己思考25分钟。

|:------|:------|

**效果：** 以前汇报前焦虑，现在用番茄钟"有步骤地准备"，焦虑少很多。

## 🤝 场景案例：人际交际

### 番茄钟社交法
- **番茄钟原则：** 和人聊天时也要"专注"
- 聊天时不要看手机（= 番茄钟不被打断）
- 如果走神了（= 番茄钟作废），道歉再重来

### 用番茄钟改善注意力
Chris 发现自己和别人说话时总走神。他练习：
- **第1个番茄钟：** 和妈妈聊天10分钟，全程不看别处
- **第2个番茄钟：** 和同学聊天15分钟，记录自己走神了几次
- **第3个番茄钟：** 专注听别人说话，不打断

**效果：** 练习一周后，聊天时走神少了很多。

## 进阶技巧

### 预估番茄数
做作业前先估算需要几个番茄钟，做完后对比：
| 作业 | 预估 | 实际 |
|:----|:----|:-----|
| 数学 | 1个🍅 | 1.5个🍅 |
| 语文 | 1个🍅 | 1个🍅 |
| 英语 | 0.5个🍅 | 1个🍅 |

**效果：** 一周后预估越来越准，时间管理能力提升。

### 记录打断
每次被打断画一个标记：
- 内部打断（自己走神了）：正字
- 外部打断（别人来找）：正字  
一周后看看哪种打断最多，针对性改进。

## 给家长的提示

- 刚开始25分钟太长，可以从**15分钟**开始，慢慢增加
- 番茄钟期间不要打扰孩子
- 休息时间让孩子**离开书桌**走动
- 不要用番茄钟做"惩罚"
- 推荐买**实体计时器**，比手机计时更专注
