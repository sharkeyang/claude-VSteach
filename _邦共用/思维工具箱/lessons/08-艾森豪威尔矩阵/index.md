# 📊 艾森豪威尔矩阵

> 第8课 · 重要/紧急四象限，分清优先级

---

## 核心思想

按**重要性**和**紧急性**两个维度，把事情分成四类。

> 大部分人整天忙"紧急的事"，忽略了"重要但不紧急的事"。  
> 正确的策略：保证"重要不紧急"的时间，这样"紧急重要"会越来越少。

---

## 四象限

| | 紧急 🔥 | 不紧急 🧊 |
|:--|:--------|:----------|
| **重要** ⭐ | **① 马上做** | **② 计划做** |
| **不重要** ➖ | **③ 委托/简化** | **④ 不要做** |

---

## 🏀 场景案例：打篮球

### 篮球训练的四象限

| 象限 | 例子 | 策略 |
|:----|:-----|:------|
| **① 重要且紧急** | 下周有比赛，每天练1小时 | 马上安排训练计划 |
| **② 重要不紧急** | 平时练投篮、运球、体能 | 每周固定时间练 |
| **③ 紧急不重要** | 同学叫你去打球（但你要做作业） | 做完作业再去 |
| **④ 不重要不紧急** | 看NBA集锦刷1小时 | 限制时间，只看10分钟 |

### 比赛前的优先级
Chris 下周有比赛，用四象限安排：

| 象限 | 内容 |
|:----|:------|
| **① 马上做** | 练投篮（命中率太低，必须马上提高） |
| **② 计划做** | 体能训练（长期重要，但不急到明天） |
| **③ 简化** | 帮同学捡球（有时间再帮） |
| **④ 不要做** | 刷手机看球鞋评测 |

## 📖 场景案例：背诵

### 背诵的四象限

| 象限 | 例子 | 策略 |
|:----|:-----|:------|
| **① 重要且紧急** | 明天要检查的古文 | 今天必须背完 |
| **② 重要不紧急** | 每天背5个单词 | 每天固定时间做 |
| **③ 紧急不重要** | 同学让你帮他背（但他明天不检查） | 让他自己背 |
| **④ 不重要不紧急** | 背课外网站的冷知识 | 有时间再看 |

### 考前复习
| 象限 | 内容 |
|:----|:------|
| **① 马上做** | 复习明天考的单元（常考题型+错题） |
| **② 计划做** | 复习下周考的单元（每天看一点） |
| **③ 简化** | 帮同学讲题（如果自己复习完了再帮） |
| **④ 不要做** | 背不考的冷门知识点 |

## 📜 场景案例：学习历史

### 学历史的四象限

| 象限 | 内容 | 策略 |
|:----|:-----|:------|
| **① 重要且紧急** | 后天历史考试 | 马上复习重点 |
| **② 重要不紧急** | 平时读历史故事书 | 每天睡前读10页 |
| **③ 紧急不重要** | 老师让查一个不重要的年份 | 快速查一下就好 |
| **④ 不重要不紧急** | 看网上"三国演义"的搞笑视频 | 不看 |

### 平时学习 vs 考前突击
**平时（第②象限）：** 每周读1章历史故事，画时间线+人物图

**考前（第①象限）：** 复习时间线+做练习题

**结果：** 平时做好了第②象限，考前第①象限的事情就少了。

## 💬 场景案例：回答问题 & 汇报

### 回答问题的四象限
老师问："谁能回答这个问题？"

| 情况 | 象限 | 策略 |
|:----|:-----|:------|
| 你确定会 | **① 马上做** | 举手回答 |
| 你不太确定，但想试试 | **② 计划做** | 先听完别人的回答，再补充 |
| 你完全不会 | **④ 不要做** | 记下来课后问 |

### 做汇报的优先级
Chris 同时有3件事：
1. **① 明天要交数学作业**（重要紧急）→ 先做
2. **② 准备下周的汇报**（重要不紧急）→ 每天准备一点
3. **③ 同学让帮忙画海报**（紧急不重要）→ 做完自己的事再帮

## 🤝 场景案例：人际交际

### 人际交往的四象限

| 象限 | 内容 | 策略 |
|:----|:-----|:------|
| **① 重要且紧急** | 和好朋友吵架了，需要马上和好 | 今天就去道歉 |
| **② 重要不紧急** | 每天和好朋友聊天、分享 | 每天课间都聊一聊 |
| **③ 紧急不重要** | 不熟的同学让你帮他值日 | 偶尔可以帮，但别耽误自己 |
| **④ 不重要不紧急** | 和同学争论"哪个游戏更好玩" | 不争了，浪费时间 |

### 交朋友的时间分配
| 象限 | 花多少时间 |
|:----|:----------|
| **① 重要且紧急** | 10%（吵架了才需要） |
| **② 重要不紧急** | 60%（每天和好朋友聊天、一起玩） |
| **③ 紧急不重要** | 20%（偶尔帮同学忙） |
| **④ 不重要不紧急** | 10%（闲聊、争论） |

## 日常练习

每晚做作业前，把今天的任务填进四象限：

```
┌──────────────────────────────────────┐
│  🔥 紧急                   🧊 不紧急 │
├──────────────────────────────────────┤
│ ⭐ ① 马上做          ② 计划做       │
│   - 数学作业(明天交)    - 练字10分钟  │
│   - 背课文(明天检查)    - 读课外书    │
├──────────────────────────────────────┤
│ ➖ ③ 简化/委托       ④ 不要做       │
│   - 回消息(做完再说)    - 刷视频      │
│   - 帮同学检查(有空)    - 发呆30分钟  │
└──────────────────────────────────────┘
```

## 给家长的提示

- 帮孩子区分"重要"和"紧急"——孩子容易把"老师催的都当重要"
- 每天花5分钟做四象限分类
- **第②象限决定了长期成长**，保证每天都有时间做
- 自己也可以用，全家一起用效果更好
