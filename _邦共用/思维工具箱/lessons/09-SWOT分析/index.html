🔍 SWOT 分析

第9课 · 分析优劣势和外部环境

核心思想： 从内部（优势/劣势）和外部（机会/威胁）四个角度全面分析决策。
做决策时最容易犯的错误：只看到好处，没看到风险；只看到自己，没看到环境。

四象限

有利的 (+)不利的 (-)
内部S 优势W 劣势
外部O 机会T 威胁

🏀 场景案例：打篮球

分析"要不要参加篮球比赛"

维度内容
S 优势跑得快、传球视野好、有3个月训练基础
W 劣势投篮命中率低（20%）、防守跟不上、容易紧张
O 机会比赛在2周后，有时间练；爸爸可以陪练
T 威胁对手去年赢了；他们有个高个子球员

策略： SO利用速度优势练快攻 → ST多传球少投篮 → WO每天练50个投篮 → WT目标定为"发挥水平"

📖 场景案例：背诵

分析"为什么背古文困难"

维度内容
S 优势记忆力不错、语文基础好、喜欢听故事
W 劣势没耐心、不理解意思就硬背、容易分心
O 机会网上有讲解视频、妈妈可以帮忙检查
T 威胁其他作业也很多、时间不够

策略： SO先看故事视频再背 → WO先看白话翻译 → ST用番茄钟每天背1段 → WT分段背，背完休息

📜 场景案例：学习历史

分析"我的历史学习"

维度内容
S 优势喜欢听历史故事、能记住有趣的事件
W 劣势记不住年份、时间线混乱
O 机会有历史纪录片、老师发了复习提纲
T 威胁考试要考很多年份、其他科目也要复习

策略： SO看纪录片建立故事感 → WO用提纲画时间线 → ST把年份编成故事 → WT每天只花15分钟画时间线

💬 场景案例：演讲比赛

维度内容
S 优势词汇量大、发音不错
W 劣势经验少、容易紧张
O 机会暑假有时间、爸爸可以辅导
T 威胁其他选手更有经验、输了可能打击信心

策略： SO准备有深度的演讲稿 → ST用发音优势弥补经验 → WO多练模拟演讲 → WT目标定为"积累经验"

🤝 场景案例：人际交际

SWOT分析"交新朋友"

维度内容
S 优势性格开朗、喜欢帮助别人、会打篮球
W 劣势有时说话太直接、不太会找话题
O 机会新同学转学来了、下周有篮球比赛
T 威胁他可能已经有朋友了、他可能不喜欢篮球

策略： SO主动打招呼 → WO约他一起打篮球 → ST先问他喜欢什么 → WT先观察不要太热情

家庭应用场景

场景怎么用
选兴趣班分析自己的优劣势，对比不同班的机会
考试目标分析各科强弱，定合理的复习策略
比赛/竞选分析自己和竞争对手，制定策略
暑假计划分析时间和资源，制定可行计划
交朋友分析自己的社交优劣势，找到突破口

给家长的提示

  • SWOT 不要只看"优劣势"——机会和威胁往往更关键
  • 帮孩子区分"内部"和"外部"
  • 分析完要制定策略
  • 定期做（每学期一次），对比前后变化
