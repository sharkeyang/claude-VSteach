# 🔍 SWOT 分析

> 第9课 · 分析优劣势和外部环境

---

## 核心思想

从**内部**（优势/劣势）和**外部**（机会/威胁）四个角度全面分析一个决策。

> 做决策时最容易犯的错误：只看到好处，没看到风险；只看到自己，没看到环境。

---

## 四象限

| | 有利的 (+) | 不利的 (-) |
|:--|:----------|:-----------|
| **内部** | **S** 优势（Strength） | **W** 劣势（Weakness） |
| **外部** | **O** 机会（Opportunity） | **T** 威胁（Threat） |

---

## 🏀 场景案例：打篮球

### 分析"要不要参加篮球比赛"

| 维度 | 内容 |
|:----|:------|
| **S 优势** | 跑得快、传球视野好、有3个月训练基础 |
| **W 劣势** | 投篮命中率低（20%）、防守跟不上、容易紧张 |
| **O 机会** | 比赛在2周后，有时间练；爸爸可以陪练；教练会指导 |
| **T 威胁** | 对手去年赢了比赛；他们有个高个子球员 |

### 策略
| 策略 | 内容 |
|:----|:------|
| **SO** | 利用跑得快优势（S）+ 2周时间（O）→ 练快攻战术 |
| **ST** | 传球视野好（S）+ 对手高个子（T）→ 多传球，少投篮 |
| **WO** | 投篮差（W）+ 爸爸陪练（O）→ 每天练50个投篮 |
| **WT** | 容易紧张（W）+ 对手强大（T）→ 目标定为"发挥水平"，不看重输赢 |

## 📖 场景案例：背诵

### 分析"为什么背古文困难"

| 维度 | 内容 |
|:----|:------|
| **S 优势** | 记忆力不错、语文基础好、喜欢听故事 |
| **W 劣势** | 没耐心、不理解古文意思就硬背、容易分心 |
| **O 机会** | 网上有古文讲解视频、妈妈可以帮忙检查 |
| **T 威胁** | 其他作业也很多、时间不够 |

### 策略
| 策略 | 内容 |
|:----|:------|
| **SO** | 喜欢听故事（S）+ 有讲解视频（O）→ 先看故事视频再背 |
| **WO** | 不理解意思（W）+ 有讲解视频（O）→ 先看白话翻译再背 |
| **ST** | 记忆力好（S）+ 时间不够（T）→ 用番茄钟，每天背1段 |
| **WT** | 没耐心（W）+ 作业多（T）→ 分段背，背完一段休息一下 |

## 📜 场景案例：学习历史

### 分析"我的历史学习"

| 维度 | 内容 |
|:----|:------|
| **S 优势** | 喜欢听历史故事、能记住有趣的事件 |
| **W 劣势** | 记不住年份、时间线混乱、不想背 |
| **O 机会** | 有历史纪录片可以看、老师发了复习提纲 |
| **T 威胁** | 历史考试要考很多年份、其他科目也要复习 |

### 策略
| 策略 | 内容 |
|:----|:------|
| **SO** | 喜欢听故事（S）+ 有纪录片（O）→ 看纪录片建立"故事感" |
| **WO** | 记不住年份（W）+ 有复习提纲（O）→ 用提纲画时间线 |
| **ST** | 喜欢故事（S）+ 考试要考年份（T）→ 把年份编成故事记忆 |
| **WT** | 时间线混乱（W）+ 其他科目也要复习（T）→ 每天只花15分钟画时间线 |

## 💬 场景案例：回答问题 & 汇报

### 用SWOT分析"要不要参加演讲比赛"

| 维度 | 内容 |
|:----|:------|
| **S 优势** | 词汇量大、发音不错、有英语基础 |
| **W 劣势** | 经验少、容易紧张、没有演讲技巧 |
| **O 机会** | 暑假有时间准备、爸爸可以帮忙辅导 |
| **T 威胁** | 其他选手更有经验、输了可能打击信心 |

### 策略
| 策略 | 内容 |
|:----|:------|
| **SO** | 词汇量大（S）+ 暑假有时间（O）→ 准备一篇有深度的演讲稿 |
| **ST** | 发音好（S）+ 其他选手有经验（T）→ 用发音优势弥补经验不足 |
| **WO** | 容易紧张（W）+ 有时间（O）→ 多练几次模拟演讲 |
| **WT** | 经验少（W）+ 对手强（T）→ 目标定为"积累经验"，不看重名次 |

## 🤝 场景案例：人际交际

### SWOT分析"交新朋友"

| 维度 | 内容 |
|:----|:------|
| **S 优势** | 性格开朗、喜欢帮助别人、会打篮球 |
| **W 劣势** | 有时说话太直接、不太会找话题 |
| **O 机会** | 新同学转学来了、下周有篮球比赛可以一起参加 |
| **T 威胁** | 他可能已经有朋友了、他可能不喜欢打篮球 |

### 策略
| 策略 | 内容 |
|:----|:------|
| **SO** | 开朗（S）+ 新同学来了（O）→ 主动打招呼，介绍自己 |
| **WO** | 不会找话题（W）+ 篮球比赛（O）→ 约他一起打篮球 |
| **ST** | 会打篮球（S）+ 他可能不喜欢篮球（T）→ 先问他喜欢什么 |
| **WT** | 说话太直接（W）+ 他可能有朋友了（T）→ 先观察，不要一上来太热情 |

## 家庭应用场景

| 场景 | 怎么用 |
|:----|:-------|
| 选兴趣班 | 分析自己的优劣势，对比不同班的机会 |
| 考试目标 | 分析各科强弱，定合理的复习策略 |
| 比赛/竞选 | 分析自己和竞争对手，制定策略 |
| 暑假计划 | 分析时间和资源，制定可行计划 |
| 交朋友 | 分析自己的社交优劣势，找到突破口 |
| 解决矛盾 | 分析矛盾的内部和外部因素 |

---

## 给家长的提示

- SWOT 不要只看"优劣势"——**机会和威胁往往更关键**
- 帮孩子区分"内部"和"外部"——不要把所有问题都归咎于自己
- 分析完要**制定策略**——SWOT 本身不是目的，策略才是
- 定期做（每学期一次），对比前后变化
