# 🔄 复盘方法论

> 第10课 · 从经验中提炼规律

---

## 来源

围棋术语，联想柳传志引入管理领域

## 核心思想

**经验本身不会带来进步，对经验的反思才会。**

> 做10年同样的事不一定会变厉害。  
> 但如果每件事做完都花10分钟复盘，进步一定比别人快。

---

## 四步复盘法

### ① 回顾目标
当初的目标是什么？期望的结果是什么？

### ② 评估结果
实际结果怎么样？和目标差多少？

### ③ 分析原因
为什么会有这个差距？成功/失败的原因是什么？

### ④ 总结规律
学到了什么？提炼出可以复用的经验，制定改进计划。

---

## 🏀 场景案例：打篮球

### 比赛复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 投进10个球，发挥出训练水平 |
| **② 评估结果** | 投了10中3，3次助攻，2次失误；输了8分 |
| **③ 分析原因** | 投篮命中率低（紧张导致姿势变形）；防守时跟不住人 |
| **④ 总结规律** | 比赛和训练不一样，比赛时心跳加快，动作会变形 → 下次训练时加入"模拟比赛"环节 |

### 投篮训练复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 今天练50个投篮，目标命中20个（40%） |
| **② 评估结果** | 投了50中18（36%），差2个 |
| **③ 分析原因** | 前20个进了10个（50%），后30个只进了8个（27%）——体力下降后命中率暴跌 |
| **④ 总结规律** | 体力下降时投篮姿势容易变形 → 以后训练时加入"疲劳投篮"练习，模拟比赛后半段的情况 |

## 📖 场景案例：背诵

### 古文背诵复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 3天背完《出师表》前两段 |
| **② 评估结果** | 3天背完了，但检查时发现中间有3句卡顿 |
| **③ 分析原因** | 卡顿的3句都是"长句+不常见字"；背的时候每段背了5遍，但没专门练长句 |
| **④ 总结规律** | 长句需要单独练，不常见的字要先查字典 → 以后背诵先把"长句+生字"挑出来重点练 |

### 英语单词背诵复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 本周背30个新单词，每天记5个 |
| **② 评估结果** | 周日检查时，记住22个（73%），忘了8个 |
| **③ 分析原因** | 忘了的8个都是"长的、拼写复杂的"；每天只背不复习，前几天的忘了 |
| **④ 总结规律** | 背单词需要"复习+新学"结合 → 以后每天先复习前一天的，再背新的 |

## 📜 场景案例：学习历史

### 历史考试复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 历史考90分以上 |
| **② 评估结果** | 考了82分，丢了18分 |
| **③ 分析原因** | 时间线题错了3道（6分）：把"官渡之战"和"赤壁之战"的顺序搞反了；原因题错了2道（8分）：只写了"是什么"没写"为什么"；选择题错了2道（4分）：粗心 |
| **④ 总结规律** | 时间线要用"画图法"来记，不能光靠背；原因题要用"因为……所以……"的句式回答 → 以后学历史先画时间线+因果链 |

### 学历史复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 一周学完"三国"章节 |
| **② 评估结果** | 读完了，但讲不出来完整故事 |
| **③ 分析原因** | 只读没练，没用自己的话复述；事件太多，没有重点 |
| **④ 总结规律** | 学历史 = 读 + 画时间线 + 讲给别人听 → 以后每学完一章，先画图再讲给爸爸听 |

## 🧮 场景案例：深度思考

### 学而思数学复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 做完5道分数应用题，全对 |
| **② 评估结果** | 错了2道 |
| **③ 分析原因** | 错的两道都是"单位1"找错了；因为题目中"甲比乙多1/3"，我搞不清"乙"是单位1 |
| **④ 总结规律** | "比"字后面的就是单位1 → 以后做题先圈出"比"字后面的量，标上"单位1" |

## 💬 场景案例：回答问题 & 汇报

### 课堂回答复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 语文课上老师提问，我想回答 |
| **② 评估结果** | 举手了，但回答时结结巴巴，说了半句就停了 |
| **③ 分析原因** | 太紧张，没想好就举手了；回答时没有"先说结论再说理由"的结构 |
| **④ 总结规律** | 回答前先在脑子里默念一遍 → 以后想清楚了再举手，用"总-分"结构 |

### 汇报复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | "我的暑假"汇报，讲3分钟，清晰连贯 |
| **② 评估结果** | 讲了2分半，内容还行，但手一直在抖，不敢看同学 |
| **③ 分析原因** | 在家练了2遍，但都是坐着练的；上台后发现站着会紧张；没有练习"看观众" |
| **④ 总结规律** | 汇报练习要"模拟真实场景" → 以后站着练，让妈妈当观众，练眼神交流 |

## 🤝 场景案例：人际交际

### 和朋友吵架复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 和XX好好相处，不吵架 |
| **② 评估结果** | 今天又因为"谁先玩"吵起来了 |
| **③ 分析原因** | 每次都是因为"抢东西"吵；没有提前约定规则；情绪上来时控制不住 |
| **④ 总结规律** | 容易吵架的事要提前定规则 → 以后和XX约好"10分钟轮换制"，用计时器 |

### 交朋友复盘
| 步骤 | 内容 |
|:----|:------|
| **① 回顾目标** | 和新同学XX成为朋友 |
| **② 评估结果** | 聊了天，但没加微信，下周可能又生疏了 |
| **③ 分析原因** | 只聊了一次，没有建立"共同活动"；没有交换联系方式 |
| **④ 总结规律** | 交朋友需要"共同经历"来巩固 → 以后约他一起打球/做值日，建立持续互动 |

## 复盘时机

| 时机 | 频率 | 用时 | 重点 |
|:----|:-----|:-----|:------|
| 每次训练/练习 | 每次 | 5分钟 | 命中率、动作标准 |
| 每次考试/比赛 | 每次 | 10分钟 | 成绩分析、策略调整 |
| 每次背诵 | 每次 | 5分钟 | 卡顿处、遗忘点 |
| 每次汇报 | 每次 | 5分钟 | 内容、节奏、紧张度 |
| 每周 | 每周 | 10分钟 | 本周整体情况 |
| 每月 | 每月 | 15分钟 | 月度总结、趋势分析 |

---

## 复盘 vs 总结

| | 总结 | 复盘 |
|:--|:-----|:-----|
| 时间 | 做完后 | 做完后+过程中 |
| 重点 | 结果 | 过程和原因 |
| 目的 | 记录 | 改进 |
| 态度 | "结束了" | "下次更好" |

---

## 给家长的提示

- 复盘不是批评大会——**语气要中立，目的是进步**
- 先让孩子自己说，家长再补充
- 分析原因是核心——不要只停留在"评估结果"
- 总结规律要**可执行**——"下次要检查"不如"做完留5分钟检查"
- 孩子不想复盘时不要强迫——5分钟就够了
