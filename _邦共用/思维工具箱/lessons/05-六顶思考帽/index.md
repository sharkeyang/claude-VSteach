# 🎨 六顶思考帽（de Bono）

> 第5课 · 一次只从一个角度思考

---

## 提出者

**Edward de Bono**，创意思维大师

## 核心思想

一次只从一个角度思考，把不同角度分开处理——避免讨论混乱和情绪化决策。

> 大部分人讨论时各种角度混在一起：事实、感觉、风险、好处……结果变成吵架。  
> 六顶帽子让每个人**同一时间从同一角度思考**，最后再综合。

---

## 六顶帽子详解

| 帽子 | 颜色 | 角度 | 典型提问 |
|:----|:-----|:-----|:---------|
| ⚪ **白帽** | 白色 | 事实与数据 | 我们有哪些信息？还缺什么？ |
| 🔴 **红帽** | 红色 | 直觉与情感 | 你现在的感觉是什么？ |
| ⚫ **黑帽** | 黑色 | 风险与问题 | 哪里可能出问题？ |
| 🟡 **黄帽** | 黄色 | 价值与收益 | 有什么好处？值得做吗？ |
| 🟢 **绿帽** | 绿色 | 创意与可能 | 还有其他方案吗？ |
| 🔵 **蓝帽** | 蓝色 | 过程与组织 | 现在讨论到哪了？下一步？ |

---

## 🏀 场景案例：打篮球

### 用六顶帽子分析"要不要参加篮球比赛"

| 帽子 | 内容 |
|:----|:------|
| ⚪ **白帽** | 比赛在2周后，每天可以练1小时；对手是隔壁班，他们去年赢了 |
| 🔴 **红帽** | 感觉：兴奋！想打！但有点紧张 |
| ⚫ **黑帽** | 风险：如果输了会不会丢脸？每天练1小时会不会影响学习？ |
| 🟡 **黄帽** | 收益：可以检验练习成果，积累比赛经验，和队友配合 |
| 🟢 **绿帽** | 创意：可以每天和队友一起练，互相配合；输了就当作"为明年积累经验" |
| 🔵 **蓝帽** | 结论：参加！目标不是赢，是"发挥出自己的水平" |

### 用六顶帽子分析比赛后的表现
| 帽子 | 内容 |
|:----|:------|
| ⚪ **白帽** | 投了10中3，3次助攻，2次失误 |
| 🔴 **红帽** | 感觉：有点遗憾，但也很开心能上场 |
| ⚫ **黑帽** | 问题：投篮命中率太低，防守时跟不上 |
| 🟡 **黄帽** | 亮点：有3次助攻，传球视野不错 |
| 🟢 **绿帽** | 改进：增加投篮练习，练习防守步伐 |
| 🔵 **蓝帽** | 总结：投篮和防守需要加强，传球是优势 |

## 📖 场景案例：背诵古文

### 用六顶帽子分析"怎么背古文更有效"
| 帽子 | 内容 |
|:----|:------|
| ⚪ **白帽** | 现在背一篇《短歌行》需要3天，但3天后会忘一部分 |
| 🔴 **红帽** | 感觉：背古文很枯燥，不想背 |
| ⚫ **黑帽** | 风险：死记硬背很快会忘，效率低 |
| 🟡 **黄帽** | 收益：背熟了能用在作文里，老师会表扬 |
| 🟢 **绿帽** | 创意：先听古文的音频故事，理解背景再背；或者用画画法，把每句诗画出来 |
| 🔵 **蓝帽** | 决定：先听1遍故事，理解意思，再分段背。每天背1段，用"画面记忆法" |

## 📜 场景案例：学习历史

### 用六顶帽子分析"秦始皇焚书坑儒"
| 帽子 | 内容 |
|:----|:------|
| ⚪ **白帽** | 事实：秦始皇下令烧毁百家书，活埋了460多名儒生 |
| 🔴 **红帽** | 感觉：很残忍，太过分了 |
| ⚫ **黑帽** | 坏处：毁灭了大量文化，很多先秦典籍失传了 |
| 🟡 **黄帽** | 角度：统一思想有助于巩固统治，减少混乱 |
| 🟢 **绿帽** | 思考：如果我是秦始皇，我会怎么做？统一思想还有其他方式吗？比如"指定官方教材" |
| 🔵 **蓝帽** | 综合：焚书坑儒确实残酷，但从秦始皇的角度是为了统一思想。不过方式太极端，有更好的方法 |

### 用六顶帽子分析"历史人物"
讨论"曹操是好人还是坏人"：
| 帽子 | 内容 |
|:----|:------|
| ⚪ **白帽** | 曹操挟天子以令诸侯，统一北方，推行屯田制 |
| 🔴 **红帽** | 感觉：觉得他很厉害，但又觉得他有点坏 |
| ⚫ **黑帽** | 问题：杀了很多人，包括吕伯奢全家 |
| 🟡 **黄帽** | 价值：统一了北方，让百姓安居乐业 |
| 🟢 **绿帽** | 思考：不能用"好人/坏人"来评价历史人物，每个人都是复杂的 |
| 🔵 **蓝帽** | 结论：曹操是一个有能力的政治家，但手段残酷。历史人物不能用好坏二分法 |

## 💬 场景案例：回答问题 & 阐述见解

### 用六顶帽子回答问题
老师问："你觉得上网课好还是线下课好？"

**不用帽子（混乱）：** "上网课可以看回放，但容易走神，线下课可以和同学一起学，但路上花时间……"（想到哪说到哪 ❌）

**用六顶帽子（清晰）：**
| 帽子 | 角度 |
|:----|:------|
| ⚪ **白帽** | 上网课省了30分钟路费，但每天屏幕时间增加2小时 |
| 🔴 **红帽** | 感觉：上网课可以穿睡衣，线下课更有氛围 |
| ⚫ **黑帽** | 风险：网课容易走神，互动少 |
| 🟡 **黄帽** | 好处：网课可以回看，线下课可以和同学讨论 |
| 🟢 **绿帽** | 创意：能不能结合？一半网课一半线下？ |
| 🔵 **蓝帽** | 结论：各有好处，关键看学什么——需要讨论的课线下好，需要自学的课线上好 |

## 🤝 场景案例：人际交际

### 用六顶帽子分析"和同桌吵架了"
| 帽子 | 内容 |
|:----|:------|
| ⚪ **白帽** | 事实：他拿了我橡皮没还，我生气说"你总是这样"，他说"我就用一下" |
| 🔴 **红帽** | 感觉：很生气，觉得他不尊重我 |
| ⚫ **黑帽** | 问题：如果继续吵，会影响关系，以后同桌会很尴尬 |
| 🟡 **黄帽** | 机会：说清楚以后，他可能以后会更注意 |
| 🟢 **绿帽** | 创意：下次他借东西前先跟我说一声；或者我多备一个橡皮 |
| 🔵 **蓝帽** | 决定：心平气和地跟他说"下次用之前跟我说一声就好" |

### 家庭讨论：用六顶帽子讨论"暑假计划"
全家用同一顶帽子发言，避免吵架：
- **白帽时间：** 暑假有60天，预算5000元，想去哪里？
- **红帽时间：** 每个人说"我特别想去的地方"（只说感觉，不争论）
- **黑帽时间：** 哪里可能有问题？时间不够？太贵？
- **黄帽时间：** 每个方案的好处是什么？
- **绿帽时间：** 有没有更有创意的方案？比如"上半个月做题+下半个月出去玩"？
- **蓝帽时间：** 综合大家意见，做决定

## 家庭应用

| 场景 | 怎么用 |
|:----|:-------|
| 做决定 | "要不要报这个兴趣班"——全家一起戴一遍帽子 |
| 讨论问题 | "暑假去哪里玩"——每顶帽子轮流发言 |
| 分析冲突 | 吵架后用帽子冷静分析，不争对错 |
| 写作文 | 用不同帽子收集素材，再组织成文 |
| 回答问题 | 用帽子结构组织答案，更清晰 |
| 学历史 | 分析历史事件，从不同角度理解 |

---

## 给家长的提示

- 全家一起玩，每人戴同一颜色的帽子发言
- 戴红帽时允许说"感觉"，不要求理由
- 黑帽和黄帽要搭配：先戴黄帽再戴黑帽，避免只说负面
- 蓝帽通常由家长担任，把控节奏
- 刚开始时可以用"实物帽子"（彩色纸折的），增加趣味性
