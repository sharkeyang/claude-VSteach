# 📋 5W1H（六何分析法）

> 第6课 · 六个问题问完，事情就全清楚了

---

## 核心思想

通过六个基本问题，把一个事情从各个角度了解清楚。

> 信息不完整就下结论，是大多数错误的根源。  
> 5W1H 让你在行动前把该问的都问了。

---

## 六个问题

| 问题 | 英文 | 含义 | 例子（暑假计划） |
|:----|:----|:-----|:----------------|
| **What** | 什么 | 是什么？做什么事？ | 去北京旅游 |
| **Why** | 为什么 | 为什么要做？目的是什么？ | 看故宫、长见识 |
| **Who** | 谁 | 谁来做？涉及哪些人？ | 全家四口 |
| **Where** | 哪里 | 在哪里做？ | 北京 |
| **When** | 什么时候 | 什么时间？多长时间？ | 7月20-25日，5天 |
| **How** | 怎么做 | 用什么方法？ | 高铁+地铁，住酒店 |

---

## 🏀 场景案例：打篮球

### 用5W1H计划篮球训练

| 问题 | 训练计划 | 比赛分析 |
|:----|:---------|:---------|
| **What** | 练投篮 + 运球 | 输了比赛 |
| **Why** | 提高命中率 | 投篮命中率太低 |
| **Who** | 自己和爸爸（帮忙捡球） | 全队5人 |
| **Where** | 小区篮球场 | 学校篮球场 |
| **When** | 每天下午4:30，练40分钟 | 昨天下午第二节课 |
| **How** | 先投20个，记下命中数，再练运球绕桩 | 对方的防守比我们严密 |

### 用5W1H分析投篮问题
Chris 投篮不准，用5W1H分析：

| 问题 | 答案 |
|:----|:------|
| **What** | 投篮时球总是偏左 |
| **Why** | 手肘没有对正篮筐，出手时偏了 |
| **Who** | 自己练，爸爸录像 |
| **Where** | 在罚球线位置 |
| **When** | 一累的时候更明显 |
| **How** | 每次出手前先对准手肘，再投 |

## 📖 场景案例：背诵

### 用5W1H制定背诵计划

| 问题 | 内容 |
|:----|:------|
| **What** | 背《出师表》前两段 |
| **Why** | 下周语文课要检查 |
| **Who** | 自己背，妈妈检查 |
| **Where** | 在书桌前，不要有干扰 |
| **When** | 每天晚饭后7:00-7:20 |
| **How** | 先理解白话意思，再分段背，每段背3遍 |

### 背诵后的复盘
| 问题 | 内容 |
|:----|:------|
| **What** | 背了但有些地方忘了 |
| **Why** | 中间几段不熟 |
| **Who** | 自己 |
| **Where** | 卡在"亲贤臣，远小人"那句 |
| **When** | 背到第三段时开始卡 |
| **How** | 下次先画"背诵地图"（每段的关键词），看着关键词复述 |

## 📜 场景案例：学习历史

### 用5W1H分析历史事件（赤壁之战）

| 问题 | 答案 |
|:----|:------|
| **What** | 赤壁之战，孙刘联军 vs 曹操 |
| **Why** | 曹操想统一南方，孙刘为了自保联合 |
| **Who** | 曹操（20万大军）vs 周瑜（5万联军） |
| **Where** | 赤壁（今湖北赤壁市） |
| **When** | 公元208年 |
| **How** | 火攻：黄盖假降，带火船冲入曹营 |

### 用5W1H分析"为什么要学历史"

| 问题 | 答案 |
|:----|:------|
| **What** | 学历史就是了解过去发生了什么 |
| **Why** | 了解过去才能理解现在，避免重复错误 |
| **Who** | 每个人都应该学 |
| **Where** | 在课堂上、在书本里、在博物馆里 |
| **When** | 从小就要学，一生都在学 |
| **How** | 不背年份，而是理解"为什么"和"因果关系" |

## 🧮 场景案例：深度思考

### 用5W1H分析问题（学而思做不出来）

| 问题 | 答案 |
|:----|:------|
| **What** | 第3题不会做，关于分数应用题 |
| **Why** | 不知道"单位1"是谁 |
| **Who** | 自己，可以问爸爸 |
| **Where** | 卡在"甲比乙多1/3"这句话 |
| **When** | 做到第3题时 |
| **How** | 先画线段图，标出"单位1"再算 |

### 用5W1H做深度思考
"为什么天空是蓝色的？"

| 问题 | 答案 |
|:----|:------|
| **What** | 天空是蓝色的 |
| **Why** | 太阳光穿过大气层时，蓝光波长最短，被散射得最多 |
| **Who** | 科学家瑞利发现的光散射原理 |
| **Where** | 在大气层中 |
| **When** | 白天阳光充足时最明显 |
| **How** | 蓝光被空气分子散射，所以从各个方向看天空都是蓝色的 |

## 💬 场景案例：回答问题 & 汇报

### 用5W1H回答问题
老师问："说说你的暑假计划。"

**差回答（不完整）：** "我暑假要去姥姥家玩。"（信息太少 ❌）

**5W1H回答（完整）：** 
- **What：** 去姥姥家住一周 + 每天上午做暑假作业 + 下午打篮球
- **Why：** 想姥姥了 + 暑假作业不能拖到最后
- **Who：** 我和妈妈一起去
- **Where：** 姥姥家在乡下
- **When：** 7月20日到27日
- **How：** 坐高铁去，每天按计划表执行

**效果：** 老师一听就知道你的计划很完整 ✅

### 用5W1H做汇报
做"我的宠物"主题汇报，先用5W1H收集素材：
- **What：** 我养了一只仓鼠
- **Why：** 因为仓鼠好养，不占地方
- **Who：** 我和妈妈一起照顾
- **Where：** 养在阳台上，笼子里放木屑和跑轮
- **When：** 去年买的，已经养了8个月
- **How：** 每天喂食换水，每周清理一次笼子

**汇报成文：** 把5W1H的资料整理成"总-分-总"结构，就是一篇完整的汇报。

## 🤝 场景案例：人际交际

### 用5W1H分析人际问题
"为什么XX不跟我玩了？"

| 问题 | 答案 |
|:----|:------|
| **What** | 最近XX不主动找我玩了 |
| **Why** | 可能是我上次体育课没选他做队友，他生气了？ |
| **Who** | 我和XX，还有他新交的朋友 |
| **Where** | 主要在体育课和课间 |
| **When** | 从上周末开始，大概3天了 |
| **How** | 明天主动找他聊天，问他"最近怎么没找我玩？" |

### 用5W1H交新朋友
| 问题 | 准备 |
|:----|:------|
| **What** | 和新同学交朋友 |
| **Why** | 他是转学生，可能需要朋友 |
| **Who** | 我，他，还有其他同学 |
| **Where** | 课间的时候，在教室 |
| **When** | 明天课间 |
| **How** | 先问他"你喜欢什么动画片？"从共同爱好聊起 |

## 给家长的提示

- 孩子说"妈妈我不舒服"——用5W1H问清楚：哪里不舒服？什么时候开始的？什么感觉？
- 孩子说"作业好多"——用5W1H帮他拆解：什么作业？每科多少？先做什么？
- 5W1H 是**信息收集工具**，不是审问——语气要好奇，不要咄咄逼人
- 养成习惯后，孩子会自动用5W1H思考问题
