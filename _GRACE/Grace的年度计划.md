- Grace 需要全程陪伴，不能期待自主学习

## 四、执行机制

| 机制 | 说明 |
|---|---|
| **陪伴式学习** | 所有学习任务均需家长陪同，不能放手 |
| **表扬先行** | 所有纠错以肯定开头，避免直接批评 |
| **小步骤推进** | 任务拆细，每完成一小步给予即时表扬 |
| **美术激励** | "做完功课就可以画画"作为正向激励 |
| **哥哥带动** | 利用Grace会看哥哥榜样的特点，正向引导 |
| **月度复盘** | 每月检查各科进度，按需调整计划 |
| **单次时长** | 每科10~15分钟，穿插休息，避免疲劳 |
