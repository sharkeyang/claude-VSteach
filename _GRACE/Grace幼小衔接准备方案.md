## 二、分科目学习路径

### 语文路径

```
阶段1（基础期）→ 阶段2（积累期）→ 阶段3（冲刺期）
```

| 阶段 | 时长 | 重点 |
|:---|:---:|:---|
| **基础期** | 1-2 个月 | 拼音声母韵母+简单识字（每天 3-5 字） |
| **积累期** | 2-3 个月 | 拼音拼读训练+识字量 300+；开始古诗背诵 |
| **冲刺期** | 1-2 个月 | 综合练习+书写训练+简单阅读 |

**推荐工具：**
- 拼音：洪恩拼音 / 拼音卡片游戏
- 识字：四五快读 / 直映认字 / 悟空识字
- 古诗：小学生必背古诗 80 首（选前 15-20 首）

### 数学路径

```
阶段1（数感建立）→ 阶段2（计算熟练）→ 阶段3（综合应用）
```

| 阶段 | 时长 | 重点 |
|:---|:---:|:---|
| **数感建立** | 1 个月 | 1-100 数数、数物对应、比大小 |
| **计算熟练** | 2-3 个月 | 10 以内→20 以内加减法（进退位） |
| **综合应用** | 1-2 个月 | 图形、分类、规律、简单应用题 |

**推荐工具：**
- 计算：口算天天练 / 十格阵 / 数轴
- 逻辑：逻辑狗 / 摩比爱数学
- 综合：学而思幼小衔接系列

### 英语路径

```
阶段1（启蒙）→ 阶段2（拼读）→ 阶段3（积累）
```

| 阶段 | 时长 | 重点 |
|:---|:---:|:---|
| **启蒙** | 1 个月 | 字母名+字母音、简单词汇（颜色/数字/动物） |
| **拼读** | 2-3 个月 | CVC 拼读（cat/dog/pig）、高频词 50+ |
| **积累** | 1-2 个月 | 分级阅读（RAZ AA-B / 牛津树 1-2） |

**推荐工具：**
- 字母/拼读：Alphablocks / Jolly Phonics
- 分级阅读：RAZ / 牛津树 / 海尼曼
- 磨耳朵：SSS 儿歌 / 小猪佩奇英文版

## 三、日常时间建议

| 时间段 | 内容 | 时长 | 说明 |
|:---|:---|:---:|:---|
| 早上 | 晨读/磨耳朵 | 15-20min | 古诗/英语儿歌 |
| 上午 | 专项学习（1 科） | 20-30min | 语文或数学，以游戏化方式 |
| 下午 | 专项学习（1 科） | 20-30min | 英语或综合 |
| 晚上 | 亲子阅读+复盘 | 15-20min | 绘本阅读+当天复习 |

**原则：** 幼儿园阶段每次专注不超过 20-30 分钟，以游戏和体验为主，不刷题。每天总学习时间控制在 1-1.5h 以内。

## 四、基线测试方案

在开始学习前，建议先做基线测试：

### 语文基线
1. **识字量：** 用识字卡片测认识多少个字
2. **拼音：** 逐个测试声母/韵母/整体认读音节的认读
3. **古诗：** 能背出几首？能背到哪？
4. **书写：** 画一条直线、画圆、写自己的名字

### 数学基线
1. **数数：** 能数到几？会不会倒数？
2. **计算：** 5以内/10以内/20以内各能做几题？
3. **图形：** 认识哪些图形？
4. **逻辑：** 简单找规律题能否做？

### 英语基线
1. **字母：** 认识几个？大小写对应？
2. **词汇：** 用图片卡片测认识多少词
3. **听说：** 能不能听懂简单指令？会不会说简单句子？
