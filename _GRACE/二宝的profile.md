## 六、教学策略

| 原则 | 说明 |
|---|---|
| **当前核心：幼小衔接** | 入学前（2026年9月）集中补齐短板，纳入正轨后再拓展 |
| **不报班原则** | 🚫 **不报任何课外班**——所有学习靠自学、观看视频、家长教三种方式完成，优先自学 |
| **陪伴式学习** | Grace需要大人全程陪伴督导，不能像Chris那样放手自学 |

| **体育优先** | 🏃 跳绳、基础体能是短板，每天安排训练 |
| **语文数学基础** | 📖📐 识字、阅读、计算是幼小衔接的核心 |
| **降低单次要求** | 不求一次到位，允许重复 |
| **表扬为主，批评绕着走** | Grace特别渴望夸奖、一说就哭、批评就闹脾气。**所有纠错必须以"先肯定"开头** |
