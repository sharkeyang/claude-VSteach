第 8 课 · 浮力

人教版八年级下册 第十章 | 全章覆盖：浮力产生原因 · 阿基米德原理 · 浮沉条件 · 应用 | 教学：40 分钟

📜 起源：一声"Eureka!"震古烁今

公元前 3 世纪 · 西西里岛 · 叙拉古王国
👑 王冠之谜

希耶隆二世（King Hieron II）让金匠做了一顶纯金王冠。王冠完工后，重量和交给金匠的黄金正好一样——但国王怀疑：金匠会不会掺了银子，把剩下的黄金私吞了？

国王找来他最信任的学者——阿基米德（Archimedes），让他想办法在不破坏王冠的前提下，查出王冠是不是纯金的。

这是个天大的难题。 阿基米德想了很久都没找到办法。
🛁 浴缸里的顿悟

一天，阿基米德去公共浴场洗澡。他坐进浴缸时，水从浴缸边缘溢了出来，同时他感到自己的身体"变轻了"——好像有什么力量在托着他。

就在那一瞬间，他明白了：

"物体浸入水中排开的水量，等于物体浸入部分的体积。而物体在水中受到的向上托力（浮力），等于排开水的重量。"

他激动得跳出浴缸，光着身子跑到大街上，一路高喊：

"Εύρηκα! Εύρηκα!"

（Eureka! Eureka!——我找到了！我找到了！）

⚖️ 验证

阿基米德回家后做了实验：
  1. 把王冠浸入水中，测量溢出的水量 V冠
  2. 把同等重量的纯金浸入水中，测量溢出的水量 V金
  3. 把同等重量的纯银浸入水中，测量溢出的水量 V银
结果：V金 < V冠 < V银——王冠的体积介于纯金和纯银之间，说明它既不是纯金也不是纯银。

金匠被处死了。阿基米德原理诞生了。
🧠 这个故事告诉孩子什么？
1. 最伟大的科学发现往往来自日常观察（泡澡）
2. 一个"跨领域"的想法——把"测体积"和"测纯度"联系起来——才是真正的天才
3. 阿基米德原理的本质：浮力 = 排开液体的重量

10.1 浮力

什么是浮力？

定义：浸在液体（或气体）中的物体，受到液体（或气体）向上托起的力，叫浮力。符号 F浮，单位 N。

浮力产生的原因——压强差

回顾第 7 课：液体压强随深度增加而增大。

🏊 想象一个浸没在水中的立方体：
  • 上表面：深度小 → 水向下的压强小
  • 下表面：深度大 → 水向上的压强大
  • 前后左右：深度相同 → 压强抵消
→ 上下表面的压力差 = 浮力

公式： F浮 = F向上 − F向下
⚠️ 如果物体底部和容器底部紧密贴合呢？
比如一块沉在水底的石头，如果底部完全贴合容器底——没有水在下面，就没有向上的压力→不受浮力。
这就是为什么打捞沉船时，要在船底塞入气囊——让水进到船底下面，产生向上的浮力。

测量浮力的方法——称重法

  1. 在空气中用弹簧测力计测物体重力 G
  2. 把物体浸入液体中，读弹簧测力计示数 F
  3. F浮 = G − F
🔬 实验现象： 物体浸入水中后，弹簧测力计示数变小了——因为水在帮我们"托着"它。 物体浸入越多，示数越小——浮力越大。

10.2 阿基米德原理

实验探究：浮力与什么有关？

通过控制变量法实验，我们可以发现：

  • 浮力与液体的密度有关：同一物体浸没在水中和在盐水中，F浮不同（盐水更大）
  • 浮力与物体排开液体的体积有关：物体浸入越多，F浮越大
  • 浮力与物体浸没后的深度无关：完全浸没后，再往下沉，F浮不变
  • 浮力与物体的密度、形状无关
"浸在液体中的物体受到向上的浮力，浮力的大小等于它排开的液体所受的重力。"

F浮 = G排 = ρ液 g V排

  • ρ液 — 液体的密度（kg/m³）
  • g — 重力常量（9.8 N/kg）
  • V排 — 物体排开液体的体积（m³）
⚠️ 三个最容易错的地方：
  1. V排 不等于 V物——只有物体完全浸没时 V排 = V物；部分浸入时 V排 < V物
  2. ρ液 是液体的密度，不是物体的密度——很多孩子会搞混，写成 ρ物
  3. 下沉的物体也受浮力——铁块沉入水底仍然受浮力（除非底部完全贴合）
🧠 竞赛视角： 阿基米德原理的推导其实很简单——想象一个浸没在液体中的物体，它排开的液体体积就是它自己的体积。 排开液体的重力 = ρ液V排g。这个"排开"的概念是阿基米德最天才的洞察。

10.3 物体的浮沉条件

浸没在水中的物体，受力分析

物体浸没时，只受两个力：重力 G（向下）和 浮力 F浮（向上）。

条件运动状态原因
F浮 > G上浮浮力更大，物体被推上去
F浮 = G悬浮二力平衡，物体可以停留在液体内部任何位置
F浮 < G下沉重力更大，物体往下沉

上浮的物体最终会露出液面，V排 减小，F浮 减小，直到 F浮 = G 时——漂浮。

浮沉条件的密度表述

因为 F浮 = ρ液gV排，G = ρ物gV物，浸没时 V排 = V物，所以：

比较结果
ρ物 < ρ液上浮 → 最终漂浮
ρ物 = ρ液悬浮
ρ物 > ρ液下沉
🌊 生活例子：
🪵 木头（ρ≈0.6 g/cm³）< 水（ρ=1.0 g/cm³）→ 漂浮
🧊 冰（ρ≈0.9 g/cm³）< 水 → 漂浮（约 1/10 露出水面）
⚓ 铁（ρ≈7.8 g/cm³）> 水 → 下沉
🐟 鱼可以通过改变鱼鳔的大小来改变平均密度，实现上浮或下沉

漂浮与悬浮的区别（必考）

漂浮悬浮
位置在液面上在液体内部
V排 与 V物V排 < V物V排 = V物
ρ物 与 ρ液ρ物 < ρ液ρ物 = ρ液
受力F浮 = GF浮 = G
⚠️ 经典陷阱： 漂浮和悬浮都是"二力平衡"状态（F浮 = G），但 V排 不同！ 考试常给一个漂浮物体问"浮力等于重力吗？"——答案是"等于"，但孩子容易认为"浮在水上所以浮力大于重力"——这是错的。

10.4 浮力的应用

🚢 轮船

轮船用钢铁制成，但为什么能浮在水上？

  • 把钢铁做成空心的（船体是中空的）→ 增大了排开水的体积 → 增大浮力
  • 平均密度 ρ船 < ρ水 → 漂浮
  • 排水量：轮船满载时排开水的质量

排水量 = 船自身的质量 + 满载时货物的质量

🧪 小实验： 一团橡皮泥放进水里会沉底。把它捏成船形放进水里——浮起来了！ 同样多的橡皮泥，形状变了→V排 变了→浮力变了。 把这个实验做给孩子看，比任何解释都有用。

🛸 潜水艇

潜水艇通过改变自身重量来实现上浮和下沉。

  • 下沉：向水舱中注水 → 重力增大 → G > F浮
  • 上浮：将水舱中的水排出 → 重力减小 → G < F浮
  • 悬浮：调节水量使 G = F浮

🎈 热气球 / 飞艇

靠空气的浮力升空。加热气球内部的空气 → 空气密度变小 → 平均密度小于外部空气 → 升空。

🌊 盐水选种

把种子放到盐水中——饱满的种子密度大→下沉；干瘪的种子密度小→上浮。

🧠 竞赛视角： 浮力所有应用的核心只有一句话：通过改变 V排 或 ρ液 来改变浮力，通过改变 G 来改变浮沉状态。 轮船→改变 V排（空心），潜水艇→改变 G（注水），热气球→改变 ρ平均（加热）。
